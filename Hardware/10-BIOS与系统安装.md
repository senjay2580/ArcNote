# 10 - BIOS 与系统安装

## BIOS / UEFI 是什么？

BIOS（Basic Input/Output System）是主板上的固件程序，电脑一开机最先运行的就是它。现代电脑用的是 **UEFI**（BIOS 的升级版），但习惯上还是叫"BIOS"。

**BIOS 的作用**：
- 检测硬件是否正常（POST 自检）
- 设置启动顺序（从哪个设备启动系统）
- 调整硬件参数（风扇转速、内存频率、CPU 超频等）

## 进入 BIOS

开机后立刻反复按对应按键：

| 品牌 | 按键 |
|------|------|
| **联想** | F2 或 Fn+F2 |
| 华硕 | F2 或 Del |
| 微星 | Del |
| 技嘉 | Del |
| 惠普 | F10 |
| 戴尔 | F2 |

> 你的联想 Y7000P：开机按 **F2** 进 BIOS

## BIOS 里的关键设置

### 1. 启动顺序（Boot Order）
决定电脑从哪个设备启动系统：
```
1st Boot Device: NVMe SSD（装了系统的硬盘）
2nd Boot Device: USB Drive（U盘，装系统时用）
3rd Boot Device: Network（网络启动，一般不用）
```
装系统时，把 USB 设备调到第一位。

### 2. 启动模式
- **UEFI**：现代标准，配合 GPT 分区表（推荐）
- **Legacy/CSM**：兼容老系统，配合 MBR 分区表
- 装 Windows 10/11 一律用 **UEFI + GPT**

### 3. 安全启动（Secure Boot）
- Windows 11 要求开启
- 装 Linux 双系统时可能需要关闭
- 位置：Security → Secure Boot → Enabled/Disabled

### 4. XMP / EXPO（内存超频）
- 内存买回来默认跑基础频率（如 DDR5 跑 4800MHz）
- 开启 **XMP**（Intel）或 **EXPO**（AMD）= 跑标称频率（如 5600MHz）
- 位置：通常在 OC / Tweaker / AI Tweaker 里
- **一定要开！否则白买高频内存**

### 5. 风扇控制
- 可以调 CPU 风扇和机箱风扇的转速曲线
- 安静模式 / 标准模式 / 全速模式

## 安装操作系统

### 准备工作
1. 一个 **8GB 以上的 U 盘**
2. 另一台能上网的电脑
3. 下载微软官方工具：**Media Creation Tool**

### 制作 Windows 安装 U 盘
1. 下载 Media Creation Tool（微软官网搜索 "下载 Windows 11"）
2. 运行，选择"为另一台电脑创建安装介质"
3. 选择语言、版本
4. 选择 U 盘，等待写入完成

> 也可以用 **Rufus** 工具 + Windows ISO 镜像制作（更灵活）

### 安装步骤
1. 插入安装 U 盘
2. 开机进 BIOS，把 U 盘设为第一启动项
3. 保存退出（F10），电脑从 U 盘启动
4. 进入 Windows 安装界面
5. 选择语言和区域 → 下一步
6. 选择"自定义安装"
7. **分区**：
   - 新装机：删除所有分区，选中"未分配空间"，点"下一步"
   - Windows 会自动创建必要分区
   - 如果有多个硬盘，选对目标硬盘！
8. 等待安装完成，自动重启
9. 拔掉 U 盘
10. 进入系统设置向导（账户、网络、隐私等）

### 安装完成后必做
1. **安装驱动**：
   - 去主板/笔记本官网下载驱动包
   - 或者用 **驱动总裁** / **Driver Booster** 一键安装
   - 显卡驱动：去 NVIDIA / AMD 官网下最新版
2. **Windows Update**：设置 → 更新 → 检查更新
3. **激活 Windows**：设置 → 系统 → 激活

## 硬盘分区建议

| 分区 | 建议大小 | 用途 |
|------|----------|------|
| C 盘 | 150-256GB | 系统 + 常用软件 |
| D 盘 | 剩余空间 | 游戏 / 文件 / 项目 |

> 现代观点：如果只有一块 SSD，不分区也可以。分区的好处是重装系统不影响 D 盘数据。

## 双系统（Windows + Linux）

简要步骤：
1. 先装 Windows
2. 压缩出一块空闲分区
3. 用 Linux 安装 U 盘启动
4. 安装 Linux 到空闲分区
5. Linux 安装程序会自动配置引导（GRUB）

> 装双系统前建议备份重要数据
