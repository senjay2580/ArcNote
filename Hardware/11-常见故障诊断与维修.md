# 11 - 常见故障诊断与维修

## 排障的基本思路

```
1. 观察现象 → 2. 缩小范围 → 3. 最小化排查 → 4. 逐一排除
```

**黄金法则**：先软件后硬件，先简单后复杂

## 开机完全无反应

### 现象：按电源键没有任何反应

排查顺序：
1. **电源线插好了吗？** 电源后面的开关打开了吗？
2. **换个插座试试**
3. **短接开机**：用螺丝刀碰触主板上的 POWER SW 两个针脚
   - 如果能开机 → 机箱开机按钮或连线坏了
   - 如果不能 → 继续排查
4. **拔掉所有外设**，只留 CPU + 1 条内存 + 电源
5. **换电源试试**（借一个或用已知好的电源测试）
6. 以上都不行 → 主板可能坏了

## 开机有电但无画面

### 现象：风扇转，但显示器没信号

排查顺序：
1. **显示器线接对了吗？** 有独显 → 必须接显卡的输出口
2. **显示器本身有问题吗？** 换一台或换条线试试
3. **内存重新拔插**（最常见原因！）
   - 拔出内存，用橡皮擦金手指，重新插好
4. **只留一条内存**，逐个插槽测试
5. **显卡重新拔插**，确认供电线插好
6. **听主板蜂鸣器报警声**（如果有装喇叭）：
   - 连续短鸣 → 内存问题
   - 一长两短 → 显卡问题
   - 持续长鸣 → 内存未检测到
7. **看主板 Debug 灯**（中高端主板有）：
   - CPU 灯亮 → CPU 问题
   - DRAM 灯亮 → 内存问题
   - VGA 灯亮 → 显卡问题
   - BOOT 灯亮 → 启动设备问题

## 蓝屏（BSOD）

### 常见蓝屏代码

| 代码 | 常见原因 |
|------|----------|
| WHEA_UNCORRECTABLE_ERROR | CPU/内存超频不稳定、硬件故障 |
| IRQL_NOT_LESS_OR_EQUAL | 驱动问题 |
| CRITICAL_PROCESS_DIED | 系统文件损坏 |
| PAGE_FAULT_IN_NONPAGED_AREA | 内存问题 |
| KERNEL_DATA_INPAGE_ERROR | 硬盘问题 |
| VIDEO_TDR_FAILURE | 显卡驱动/硬件问题 |

### 蓝屏排查步骤
1. 记住蓝屏代码（拍照）
2. 如果刚装了驱动/软件 → 卸载它
3. 运行 `sfc /scannow` 修复系统文件
4. 运行内存检测（Windows 内存诊断工具 或 MemTest86）
5. 检查硬盘健康（CrystalDiskInfo）
6. 更新/回滚显卡驱动
7. 反复蓝屏 → 考虑重装系统

## 电脑频繁死机/卡顿

### 排查方向

**软件原因**：
- 任务管理器看 CPU/内存/硬盘 占用率
- 硬盘 100% 占用 → 可能是 HDD 太慢或 SSD 掉速
- 内存不够 → 加内存
- 病毒/流氓软件 → 杀毒扫描

**硬件原因**：
- **过热**：用 HWMonitor 查温度，CPU >95°C 需要清灰/换硅脂
- **内存故障**：跑 MemTest86 测试
- **硬盘故障**：CrystalDiskInfo 查健康状态，"警告"或"差"需要换盘
- **电源不稳**：高负载时死机/重启 → 电源功率不够或老化

## WiFi / 蓝牙 消失

这个你亲身经历过！参考你的 [无线网卡问题描述](./无线网卡问题描述.txt)。

### 常见原因
1. **驱动问题**：设备管理器里卸载重装驱动
2. **硬件松动**：M.2 WiFi 卡或天线接头松了（你的情况很可能是这个）
3. **BIOS 设置**：BIOS 里 WLAN 模块被禁用了
4. **系统问题**：网络重置（设置 → 网络 → 高级网络设置 → 网络重置）

### 你的 Y7000P 的 WiFi 问题
根据你的记录，按内置键盘就会触发 WiFi 消失，外接键盘不会。这强烈指向：
- **M.2 WiFi 卡（AX211）松动**：按键时机身轻微形变导致接触不良
- **u.FL 天线接头脱落**：天线线缆从网卡上松脱
- 解决方案：打开后盖，重新插紧 WiFi 卡和天线接头

## 电脑噪音大

| 噪音类型 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 持续呼呼声 | 风扇积灰/轴承老化 | 清灰，严重就换风扇 |
| 高频滋滋声 | 电感啸叫（coil whine） | 正常现象，严重可换电源/显卡 |
| 咔咔声 | 机械硬盘磁头异常 | 备份数据！可能快坏了 |
| 金属摩擦声 | 风扇叶片碰到线缆 | 整理线材 |

## 常用诊断工具

| 工具 | 用途 |
|------|------|
| **CPU-Z** | 查看 CPU/内存/主板详细信息 |
| **GPU-Z** | 查看显卡详细信息 |
| **HWMonitor** | 监控温度、电压、风扇转速 |
| **CrystalDiskInfo** | 查看硬盘健康状态 |
| **CrystalDiskMark** | 测试硬盘读写速度 |
| **MemTest86** | 内存稳定性测试 |
| **FurMark** | 显卡烤机测试 |
| **AIDA64** | 全面硬件信息 + 烤机 |
| **鲁大师** | 中文界面，适合入门 |
