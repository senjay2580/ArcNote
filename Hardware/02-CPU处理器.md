# 02 - CPU 处理器

## CPU 是什么？

CPU（Central Processing Unit，中央处理器）是电脑的"大脑"，负责执行所有指令和计算。

## 核心参数（买 CPU / 看配置必知）

| 参数 | 含义 | 举例 |
|------|------|------|
| **核心数** | 几个独立的处理单元 | 6核、8核、10核 |
| **线程数** | 逻辑处理单元（超线程技术让1核当2用） | 12线程、16线程 |
| **主频** | 每秒运算的时钟周期数，单位 GHz | 基础 2.4GHz / 睿频 4.9GHz |
| **缓存** | CPU 内部的超高速存储（L1/L2/L3） | L3 缓存 24MB |
| **制程** | 芯片制造工艺，越小越先进 | 7nm、5nm、4nm |
| **TDP** | 热设计功耗，散热器需要压住的热量 | 65W、125W |
| **插槽** | CPU 安装到主板的接口类型 | LGA 1700、AM5 |

## Intel vs AMD（2024-2025 主流）

### Intel 命名规则
```
Intel Core i7 - 13620H
       |    |    |   |
       |    |    |   └─ 后缀：H=高性能笔记本 K=可超频 F=无核显 U=低功耗
       |    |    └───── 620：具体型号
       |    └────────── 13：第13代
       └─────────────── i7：定位（i3<i5<i7<i9）
```

最新一代：Intel Core Ultra（Meteor Lake / Arrow Lake）

### AMD 命名规则
```
AMD Ryzen 7 7800X3D
          |  |   |  |
          |  |   |  └─ 3D：3D V-Cache 大缓存
          |  |   └──── X：高性能版
          |  └──────── 7800：第7代 800系列
          └─────────── 7：定位（3<5<7<9）
```

## CPU 长什么样？

### 台式机 CPU
- 一块方形金属盖的芯片，底部有密密麻麻的触点或针脚
- **Intel LGA**：触点在 CPU 上，针脚在主板上
- **AMD AM4/AM5**：PGA（针脚在 CPU 上）/ LGA（新 AM5 针脚在主板上）

### 笔记本 CPU
- 直接焊在主板上（BGA封装），不可更换
- 你的联想 Y7000P 的 i7-13620H 就是焊死的

## 安装 CPU（台式机）

1. 打开主板 CPU 插槽的金属压杆
2. 对准 CPU 角上的三角标记（防呆设计）
3. 轻轻放入，**绝对不要用力按压**
4. 放下压杆锁紧
5. 涂硅脂 → 安装散热器

> **注意**：CPU 底部的针脚/触点极其脆弱，弯了基本报废。拿 CPU 时捏边缘，不要碰底部！

## CPU 与你的电脑

你的 **i7-13620H**：
- 10 核 16 线程（6 性能核 + 4 能效核）
- 基础频率 2.4GHz，睿频 4.9GHz
- 24MB L3 缓存
- BGA 封装（焊死在主板上）
