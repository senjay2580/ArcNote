# 06 - 显卡 GPU

## 显卡是什么？

显卡（Graphics Card）的核心是 GPU（图形处理器），专门处理图像渲染和并行计算。

## 独立显卡 vs 核芯显卡

| 对比 | 独立显卡（独显） | 核芯显卡（核显） |
|------|----------------|----------------|
| 位置 | 单独一块板卡，插在 PCIe 插槽 | 集成在 CPU 内部 |
| 性能 | 强大 | 较弱，日常办公够用 |
| 功耗 | 高（几十到几百瓦） | 低（几瓦到十几瓦） |
| 价格 | 几百到几万元 | 包含在 CPU 价格里 |
| 适用 | 游戏/设计/AI | 办公/视频/轻度游戏 |

> 注意：Intel 带 F 后缀的 CPU（如 i5-13400F）没有核显，必须配独显

## NVIDIA vs AMD

### NVIDIA（N卡）命名规则
```
NVIDIA GeForce RTX 4070 Ti SUPER
              |    |   |   |
              |    |   |   └── SUPER = 加强版
              |    |   └────── Ti = 性能增强版
              |    └────────── 4070 = 40系列 70级别
              └─────────────── RTX = 支持光线追踪（GTX = 老一代）
```

**定位排序**（40系）：4060 < 4060Ti < 4070 < 4070Ti < 4080 < 4090

### AMD（A卡）命名规则
```
AMD Radeon RX 7800 XT
                |    |
                |    └── XT = 高性能版
                └────── 7800 = 7000系列 800级别
```

## 显卡的关键参数

| 参数 | 含义 | 举例 |
|------|------|------|
| **显存（VRAM）** | 显卡自己的内存 | 8GB / 12GB / 16GB / 24GB |
| **显存类型** | 显存的代数 | GDDR6 / GDDR6X |
| **核心频率** | GPU 运行速度 | 2520 MHz |
| **CUDA 核心数** | NVIDIA 并行计算单元数 | 5888 个 |
| **功耗（TGP）** | 整卡功率 | 200W / 320W |
| **供电接口** | 需要的外接电源线 | 8pin / 12VHPWR |

## 显卡长什么样？（台式机独显）

```
┌──────────────────────────────────┐
│  [风扇] [风扇] [风扇]            │ ← 散热风扇（1-3个）
│  ════════════════════            │ ← 散热鳍片
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓           │ ← PCB板（核心、显存在上面）
│  ──────────────────              │
│  [PCIe金手指]                    │ ← 插入主板 PCIe x16 插槽
└──┤HDMI├┤DP├┤DP├┤DP├─────────────┘
   └────视频输出接口────┘
```

## 安装独立显卡（台式机）

1. 拆掉机箱后方对应的 PCIe 挡板（1-3 个）
2. 对准主板 PCIe x16 插槽，竖直插入
3. 听到"咔嗒"声，插槽卡扣锁住
4. 用螺丝固定到机箱
5. 接电源线（6pin / 8pin / 12VHPWR）
6. 安装显卡驱动

> **注意**：高端显卡很重（1-2kg），建议用显卡支架防止 PCIe 插槽受力弯曲

## 笔记本独显

- 焊死在主板上，不可更换
- 功耗和性能低于同型号台式机版本
- 你的 Y7000P 配的 RTX 4050/4060 Laptop 就是笔记本版

## 显示输出接口

| 接口 | 特点 |
|------|------|
| **HDMI 2.1** | 通用，支持 4K 120Hz |
| **DisplayPort 2.0** | 专业，支持更高分辨率/刷新率 |
| **USB-C (DP Alt)** | Type-C 口输出视频 |
| **VGA / DVI** | 老接口，基本淘汰 |
