# 09 - 装机实战指南

## 装机前准备

### 工具
- **十字螺丝刀**（必须）：Phillips #2 号，磁性的最好
- **扎带/魔术贴**：理线用
- **防静电手环**：防止静电击穿元件（没有的话，装之前摸一下金属水管放电）
- **硅脂**：散热器可能自带，没带就单独买
- **开箱刀**：拆包装

### 装机顺序（推荐）

```
1. CPU → 主板
2. 内存 → 主板
3. M.2 SSD → 主板
4. CPU 散热器 → 主板
   （以上在主板包装盒上操作，方便施力）
5. 主板 → 机箱
6. 电源 → 机箱
7. 显卡 → 主板
8. 接线（电源线 + 前面板线）
9. 理线
10. 开机测试
```

## 第一步：安装 CPU

### Intel（LGA 触点式）
1. 打开主板 CPU 插槽的金属压杆（向上掰起）
2. 掀开金属保护盖
3. 找到 CPU 和插槽上的 **三角标记**（左下角），对齐
4. 轻轻放下 CPU，**不要用力按**，靠自重即可
5. 放下金属盖，压下压杆锁紧
6. 保护盖会自动弹出（留着，以后卖主板要装回去）

### AMD AM5（同样 LGA 触点式）
- 步骤和 Intel 一样

### AMD AM4（PGA 针脚式）
- 针脚在 CPU 上，对准插槽的孔位
- 对齐三角标记后，垂直放入
- 拉下压杆锁紧

> **最重要的事**：绝对不要大力出奇迹！CPU 针脚/触点弯了 = 几百到几千块打水漂

## 第二步：安装内存

1. 打开内存插槽两端的卡扣（有的主板只有一端能打开）
2. 对准防呆缺口（只有一个方向能插）
3. 两端同时均匀用力按下
4. 听到"咔嗒"声，卡扣自动合上 = 到位

### 双通道插法
```
4个插槽时（从左到右 1234）：
推荐插 2 和 4 号位（跳着插）

 [空] [内存] [空] [内存]
  1     2     3     4
```

## 第三步：安装 M.2 SSD

1. 找到主板上的 M.2 插槽
2. 如果有散热片，拧下螺丝，拿掉散热片
3. 撕掉散热片上的导热贴保护膜（如果有）
4. SSD 斜 30° 插入 M.2 插槽
5. 按下 SSD 另一端，用螺丝固定（有些主板用免工具卡扣）
6. 盖回散热片

## 第四步：安装 CPU 散热器

### 塔式风冷
1. 安装背板到主板背面（穿过安装孔）
2. 在 CPU 表面涂硅脂（米粒大小居中一点）
3. 将散热器对准 CPU 放下
4. 对角线拧紧螺丝（先对角轻拧，再逐步拧紧）
5. 安装风扇到散热塔上
6. 风扇电源线插到主板 CPU_FAN 接口

### 一体水冷
1. 安装背板
2. 涂硅脂
3. 安装冷头到 CPU 上（拧紧螺丝）
4. 将冷排用螺丝固定到机箱顶部或前面板
5. 冷排风扇线接 SYS_FAN，水泵线接 CPU_FAN 或 AIO_PUMP

## 第五步：主板装入机箱

1. 先在机箱里安装铜柱（对应主板螺丝孔位置）
2. 安装 I/O 挡板到机箱后方（从里面卡进去）
3. 将主板斜着放入，先对齐 I/O 挡板
4. 对齐铜柱螺丝孔
5. 用螺丝固定主板（先对角拧，再逐个拧紧）

## 第六步：安装电源

1. 电源风扇朝下（机箱底部有过滤网时）
2. 从机箱背面推入电源仓位
3. 4 颗螺丝固定

## 第七步：安装显卡

1. 拆掉机箱后方对应的 PCIe 挡板
2. 对准 PCIe x16 插槽，垂直均匀按下
3. 插槽尾部卡扣锁住 = 到位
4. 用螺丝固定到机箱
5. 接显卡供电线

## 第八步：接线

### 电源线
```
24pin 主板供电    → 主板右侧长条形接口
8pin CPU 供电    → 主板左上角
PCIe 供电        → 显卡顶部/侧面
SATA 电源        → 硬盘/风扇集线器
```

### 前面板接线（最难的一步！）
```
机箱前面板引出的细线：
- POWER SW (开机键)     → 对照主板说明书插
- RESET SW (重启键)     → 对照主板说明书插
- POWER LED (电源灯)    → 注意正负极
- HDD LED (硬盘灯)      → 注意正负极
- USB 3.0 (前置USB)     → 主板上的 USB3.0 接针（19pin 蓝色）
- USB-C (前置Type-C)    → 主板上的 USB-C 接针
- HD AUDIO (前置耳机口)  → 主板上的 AAFP 接针
```

> **小技巧**：前面板接线看主板说明书的图示，或者看主板上接针旁的印字

## 第九步：理线

- 电源线从机箱背面走线（利用背板空间）
- 多余的线用扎带捆好
- 不要让线碰到风扇

## 第十步：开机测试

1. 接上显示器（接显卡输出口，不是主板！有独显的话）
2. 接上电源线
3. 接上键盘鼠标
4. 打开电源后面的开关
5. 按前面板开机键
6. 看到主板 LOGO / 进入 BIOS = 成功！

### 开机不亮？排查清单
- [ ] 电源后面的开关打开了吗？
- [ ] 24pin 和 8pin CPU 供电都插了吗？
- [ ] 内存插到位了吗？（重新拔插试试）
- [ ] 显示器线接在显卡上了吗？（不是主板）
- [ ] CPU 有没有装反？（不太可能，有防呆）
- [ ] 前面板 POWER SW 线插对了吗？（可以用螺丝刀短接试试）
