# 08 - 散热系统

## 为什么需要散热？

CPU 和 GPU 工作时会产生大量热量。温度过高会导致：
- **降频**：自动降低性能保护自己（最常见）
- **死机/蓝屏**：系统不稳定
- **缩短寿命**：长期高温加速元件老化
- **烧毁**：极端情况下硬件损坏

## 散热的基本原理

```
CPU 发热 → 硅脂传导 → 散热器吸热 → 风扇/水泵把热量排出机箱
```

## 导热硅脂（硅脂/散热膏）

硅脂涂在 CPU 和散热器之间，填充微观不平整的缝隙，帮助导热。

### 涂硅脂的方法
- **中心点法**：挤一粒米大小在 CPU 中心，散热器压上去会自然摊开（推荐）
- **X 型法**：画个 X
- **薄涂法**：用刮片均匀薄涂（适合有经验的人）

> **关键**：不要涂太多！太多反而影响导热。薄薄一层就够

### 多久换一次？
- 台式机：2-3 年换一次
- 笔记本：1-2 年换一次（散热空间小，更需要维护）
- 你的 Y7000P 用了 2-3 年，建议换硅脂了

## 散热器类型

### 1. 风冷散热器

#### 下压式（原装/入门）
```
    ═══════  ← 风扇
    ╔═════╗
    ║铝块  ║  ← 铝制散热片
    ╚══╤══╝
       │     ← 直接接触 CPU
```
- CPU 原装散热器大多是这种
- 适合低功耗 CPU（65W 以下）

#### 塔式（主流/高端）
```
    ┌───┐
    │风│
    │扇│══════  ← 大面积散热鳍片
    │  │══════
    └───┘  │
           │    ← 热管（铜管，里面有液体循环）
        [CPU]
```
- 通过热管将热量传导到大面积鳍片
- 双塔双风扇 = 最强风冷
- 推荐：利民、猫头鹰（Noctua）、九州风神

### 2. 水冷散热器

#### 一体式水冷（AIO）
```
[冷头+水泵] ——水管—— [冷排+风扇]
   装在CPU上           装在机箱顶部/前面
```
- 冷排尺寸：120mm / 240mm / 360mm
- 散热能力随冷排尺寸增大
- 安装比塔式风冷简单（不会挡内存）
- 推荐：Arctic、利民、海盗船

#### 分体式水冷（DIY）
- 自己组装水泵、水箱、冷排、水管
- 散热极强，但价格高、有漏液风险
- 非发烧友不建议

## 机箱风道

好的风道 = 好的散热效果

```
         排风 ↑（顶部/后方）
    ┌─────────────────┐
    │    冷排/排风扇   │
    │                  │
    │  CPU散热器  GPU  │
    │                  │
    │   进风扇          │
    └─────────────────┘
    进风 ↑（前面板/底部）
```

**原则**：前进后出、下进上出（热空气上升）

## 笔记本散热

笔记本散热空间极小，结构通常是：
- 铜管连接 CPU 和 GPU
- 1-2 个离心风扇（鼓风机式）
- 从底部进风，侧面/后方排风

### 笔记本散热维护
1. **定期清灰**：1-2 年拆机清理风扇和散热鳍片
2. **更换硅脂**：配合清灰一起做
3. **散热底座**：辅助降温 2-5 度
4. **不要堵出风口**：放在床上/沙发上会堵进风口

## 温度多少算正常？

| 状态 | CPU 温度 | GPU 温度 |
|------|----------|----------|
| 待机 | 35-50°C | 30-45°C |
| 日常使用 | 50-70°C | 40-60°C |
| 满载/游戏 | 70-90°C | 65-85°C |
| 危险 | >95°C | >90°C |

> 查看温度：任务管理器（GPU）、HWMonitor、鲁大师
