# DocFlow 项目评分要点总结

## 项目概述

DocFlow 是一个在线文档协作平台，基于 JFinal 框架开发，实现了完整的文档管理、用户管理、文件系统管理、文档共享、分享链接等功能。

## 数据库表结构概览

| 表名 | 说明 | 外键关联 |
|------|------|----------|
| user | 用户表 | - |
| fs_node | 文件系统节点表 | user_id → user.id |
| document_content | 文档内容表 | node_id → fs_node.id |
| share | 共享记录表 | document_id → fs_node.id, from_user_id/to_user_id → user.id |
| forward | 分享链接表 | user_id → user.id |
| forward_item | 分享内容关联表 | forward_id → forward.id, fs_node_id → fs_node.id |
