# 05 - 存储设备

## 存储设备是什么？

存储设备用来永久保存数据（断电不丢失）。主要分三大类：

## 三种主流存储设备

### 1. 机械硬盘（HDD）
```
结构：旋转的磁盘 + 读写磁头（像唱片机）
优点：容量大、便宜（1TB 约 200-300 元）
缺点：慢、怕震动、有噪音
速度：约 100-200 MB/s
用途：仓库盘（存电影、备份）
```

### 2. SATA 固态硬盘（SATA SSD）
```
结构：闪存芯片，无机械部件
优点：比 HDD 快很多、无噪音、抗震
缺点：速度受 SATA 接口限制
速度：约 500-550 MB/s（SATA 上限）
接口：SATA 或 M.2（走 SATA 协议）
用途：老电脑升级首选
```

### 3. NVMe 固态硬盘（NVMe SSD）
```
结构：闪存芯片，走 PCIe 通道
优点：极快
缺点：较贵、高速读写会发热
速度：PCIe 3.0 约 3500 MB/s
       PCIe 4.0 约 7000 MB/s
       PCIe 5.0 约 14000 MB/s
接口：M.2 插槽（走 NVMe 协议）
用途：系统盘、游戏盘
```

## 接口与外形

### M.2 接口（主流）
- 像一根口香糖大小的小板子
- **M.2 2280**：22mm宽 × 80mm长（最常见）
- **M.2 2230**：22mm宽 × 30mm长（小型设备/PS5 扩展）
- 有 **M key** 和 **B+M key** 之分：
  - M key = NVMe（快）
  - B+M key = SATA（慢）

### 2.5 英寸（传统）
- 和笔记本 HDD 一样大小
- SATA SSD 和 HDD 都用这个尺寸
- 用 SATA 数据线 + SATA 电源线连接

### 3.5 英寸
- 台式机 HDD 的标准尺寸
- 比 2.5 寸大一圈

## 安装存储设备

### 安装 M.2 SSD
1. 找到主板上的 M.2 插槽（可能有散热片盖着）
2. 拧下固定螺丝，掀开散热片
3. 30度角斜插入 M.2 插槽
4. 按下另一端，拧上螺丝固定
5. 盖回散热片

### 安装 2.5/3.5 寸硬盘
1. 把硬盘装进机箱硬盘位（用螺丝或免工具托架）
2. 接 SATA 数据线（主板 → 硬盘）
3. 接 SATA 电源线（电源 → 硬盘）

## 你的电脑

联想 Y7000P IRH8：
- 有 1-2 个 M.2 2280 NVMe 插槽
- 无 2.5 寸硬盘位（大多数新笔记本已取消）
- 可以自己更换/加装 M.2 SSD

## 选购建议

| 用途 | 推荐 |
|------|------|
| 系统盘 | 500GB-1TB NVMe SSD |
| 游戏盘 | 1-2TB NVMe SSD |
| 仓库盘 | 2-4TB HDD（台式机） |
| 老电脑升级 | 任意 SATA SSD 都能起飞 |

> 品牌推荐：三星、西部数据、致态、铠侠、海力士
