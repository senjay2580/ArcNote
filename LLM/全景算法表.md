# AI & 机器学习完整学习路线（精简版 后续可拓展）
## 一、数学基础（快速过，重直觉不重推导）
1. **线性代数**
    - 内容：矩阵、向量、特征值
    - 资源：3Blue1Brown《线性代数的本质》
2. **概率统计**
    - 内容：贝叶斯、分布、期望
    - 资源：同上系列 + 南瓜书对应章节
3. **《南瓜书》**
    - 定位：周志华《机器学习》（西瓜书）公式推导详解
    - 作用：拆解西瓜书难懂公式，2026持续更新
4. **核心数学直觉**
    - 理解梯度下降为何能实现优化即可

---

## 二、机器学习入门（2~3周）
### 学习顺序
直接从吴恩达ML笔记开始，遇到公式查阅南瓜书

### 必懂核心概念
- 训练集/测试集/验证集：数据划分方式与划分原因
- 过拟合/欠拟合：模型死记硬背 vs 没学会
- 损失函数：衡量模型误差的指标
- 梯度下降：深度学习核心，重直觉无需手推公式（看3B1B动画）
- 特征工程：数据质量 > 模型选择

### 必学6大算法（难度&理由）
| 算法         | 难度 | 必学原因                         |
| ------------ | ---- | -------------------------------- |
| 线性回归     | 低   | ML入门，理解拟合与损失的最简案例 |
| 逻辑回归     | 低   | 分类任务起点，名似回归实为分类   |
| 决策树       | 低   | 逻辑直观如if-else，结果可解释    |
| 随机森林     | 低   | 多决策树投票，工业界常用、效果稳 |
| KNN（K近邻） | 最低 | 物以类聚，零数学门槛，快速理解   |
| 朴素贝叶斯   | 低   | 垃圾邮件分类经典，概率思维入门   |

### 现阶段可跳过内容
| 内容                | 跳过原因                         |
| ------------------- | -------------------------------- |
| SVM（支持向量机）   | 数学重，已被深度学习大量替代     |
| EM算法              | 纯数学推导，入门无需掌握         |
| 隐马尔可夫模型      | NLP传统方法，现被Transformer取代 |
| 主成分分析PCA       | 了解降维概念即可，无需深究推导   |
| 数学证明/收敛性分析 | 面向研究生，工程师会用即可       |

### 实践要求
1. 用scikit-learn实现上述6种算法
2. 完成小项目：房价预测（回归）/鸢尾花分类（分类）
3. 学会查看混淆矩阵、准确率/召回率，评估模型效果

# 进阶方向选择（基础完成后）
## 方向A：大模型应用开发（最适合软件开发者，变现最快）
1. **Prompt Engineering（1周）**
    - 内容：提示词编写技巧
    - 资源：llm-cookbook
2. **RAG检索增强生成（2周）**
    - 作用：让大模型基于私有数据回答
    - 适配项目：carehelp可落地使用
3. **Agent开发（2~3周）**
    - 作用：AI自主调用工具完成任务
    - 示例：Claude Code属于Agent
4. **微调Fine-tuning（按需学习）**
    - 资源：self-llm《开源大模型食用指南》

### 里程碑
可独立开发RAG问答系统

## 方向B：计算机视觉CV
- 应用：图像/视频识别、目标检测
- 场景：安防、医疗影像、自动驾驶
- 特点：门槛高，需GPU资源
- 建议：无明确场景暂不推荐

## 方向C：自然语言处理NLP
- 应用：文本分类、情感分析、机器翻译
- 现状：基本被大模型覆盖
- 建议：直接选择方向A更实用

## 方向D：强化学习RL
- 应用：AI游戏训练、机器人控制
- 特点：数学要求极高（马尔可夫决策过程）
- 建议：兴趣驱动，不建议作为首选方向

## 全景算法表

### 一、传统机器学习算法

#### 监督学习 — 分类（给数据贴标签）

#### 监督学习 — 回归（预测连续值）

| 算法                             | 本质在做什么                | 核心思想                     | 优势                           | 局限             | 典型应用                             |
| -------------------------------- | --------------------------- | ---------------------------- | ------------------------------ | ---------------- | ------------------------------------ |
| **线性回归** (Linear Regression) | 拟合一条直线/平面           | y = wx + b，最小化误差平方和 | 最简单、可解释、计算快         | 只能拟合线性关系 | 房价预测、销量预测、经济学建模       |
| **Ridge / Lasso**                | 线性回归 + 防过拟合的惩罚项 | 加正则化约束权重大小         | 防止过拟合、Lasso 能做特征选择 | 仍是线性         | 高维小样本回归（基因数据、金融因子） |
| **多项式回归**                   | 拟合一条曲线                | 把特征做幂次展开再线性回归   | 能拟合非线性                   | 高次容易过拟合   | 物理实验曲线拟合                     |
| **SVR** (支持向量回归)           | SVM 的回归版本              | 在误差容忍带内找最平的超平面 | 高维强、对异常值鲁棒           | 大数据慢         | 股价预测、负荷预测                   |

#### 无监督学习（没有标签，自己发现结构）

| 算法                               | 本质在做什么                    | 核心思想                           | 优势                                     | 局限                     | 典型应用                                   |
| ---------------------------------- | ------------------------------- | ---------------------------------- | ---------------------------------------- | ------------------------ | ------------------------------------------ |
| **K-Means**                        | 把数据自动分成 K 堆             | 反复计算"每个点离哪个中心最近"     | 简单快速                                 | 要预设 K、只能发现球形簇 | 客户分群、图像颜色压缩、用户画像           |
| **DBSCAN**                         | 按密度找"抱团"的区域            | 高密度区域 = 一个簇，低密度 = 噪声 | 不用预设 K、能发现任意形状、自动过滤噪声 | 密度不均匀时效果差       | 地理位置聚类、异常检测                     |
| **层次聚类** (Hierarchical)        | 像族谱一样逐层合并/拆分         | 自底向上合并最近的两簇             | 可视化谱系图、不用预设 K                 | 大数据慢                 | 基因表达分析、社交网络社区发现             |
| **PCA** (主成分分析)               | 降维——把 100 个特征压缩到 10 个 | 找数据方差最大的方向投影           | 去噪、加速、可视化                       | 只能线性降维             | 数据预处理、人脸特征压缩、可视化高维数据   |
| **t-SNE / UMAP**                   | 高维数据可视化到 2D/3D          | 保留邻近关系的非线性降维           | 可视化效果极佳                           | 不可逆、计算慢           | 论文里的聚类可视化图                       |
| **关联规则** (Apriori / FP-Growth) | 发现"买A的人也会买B"            | 频繁项集挖掘 + 置信度筛选          | 可解释、直觉好                           | 高维组合爆炸             | 超市购物篮分析、交叉销售推荐               |
| **异常检测** (Isolation Forest)    | 找出"不合群"的数据点            | 异常点更容易被随机切分隔离         | 高维有效、不需要标签                     | 对局部异常不敏感         | 信用卡欺诈检测、网络入侵检测、设备故障预警 |

### 二、深度学习算法

#### 基础网络

| 算法                   | 本质在做什么                              | 核心思想                                 | 与传统 ML 的关系                         | 典型应用                                     |
| ---------------------- | ----------------------------------------- | ---------------------------------------- | ---------------------------------------- | -------------------------------------------- |
| **MLP** (多层感知机)   | 最基本的神经网络，多层线性变换 + 激活函数 | 万能近似定理：足够宽的网络能拟合任意函数 | 相当于逻辑回归/线性回归的"深度版"        | 表格数据深度学习baseline                     |
| **CNN** (卷积神经网络) | 用滑动窗口自动提取空间特征                | 局部感受野 + 权重共享 + 池化             | 替代了传统 CV 中手工设计的 SIFT/HOG 特征 | 图像分类、目标检测、医学影像、自动驾驶       |
| **RNN** (循环神经网络) | 处理序列数据，有"记忆"                    | 隐藏状态在时间步之间传递                 | 替代了传统 NLP 中的 n-gram + HMM         | 时间序列、语音识别（早期）、机器翻译（早期） |
| **LSTM / GRU**         | RNN 的改进版，解决长期记忆遗忘问题        | 门控机制控制信息的记忆/遗忘/输出         | RNN 能记住 5-10 步，LSTM 能记住 100+ 步  | 机器翻译、语音识别、股价预测                 |

#### 革命性架构

| 算法                         | 本质在做什么                 | 核心思想                                 | 为什么重要                           | 典型应用                                        |
| ---------------------------- | ---------------------------- | ---------------------------------------- | ------------------------------------ | ----------------------------------------------- |
| **Transformer**              | 用注意力机制并行处理整个序列 | Self-Attention：每个词同时关注所有其他词 | 彻底取代了 RNN，并行化训练、性能碾压 | **现代 AI 的基石**：GPT、BERT、ChatGPT 全基于此 |
| **BERT**                     | 双向理解文本语义             | 掩码语言模型 + 下一句预测，预训练后微调  | NLP 的 ImageNet 时刻                 | 搜索引擎、问答系统、文本分类、命名实体识别      |
| **GPT 系列**                 | 自回归生成文本               | 从左到右逐词预测下一个词                 | 证明了 scale up 的力量               | ChatGPT、代码生成、文章写作、对话系统           |
| **ViT** (Vision Transformer) | 把 Transformer 用到图像上    | 图像切成 patch → 当成"词"序列处理        | 证明 Transformer 不只是 NLP 专属     | 图像分类、目标检测（新SOTA）                    |

#### 生成模型

| 算法                           | 本质在做什么                         | 核心思想                         | 与其他算法的关系            | 典型应用                                 |
| ------------------------------ | ------------------------------------ | -------------------------------- | --------------------------- | ---------------------------------------- |
| **GAN** (生成对抗网络)         | 生成器造假、判别器鉴真，互相对抗进化 | 博弈论：两个网络玩零和游戏       | 最早的高质量图像生成方法    | 图像生成、风格迁移、数据增强、deepfake   |
| **VAE** (变分自编码器)         | 学习数据的潜在分布，从中采样生成     | 编码器压缩 → 潜空间 → 解码器还原 | 比 GAN 更稳定但生成质量略低 | 图像生成、药物分子生成、异常检测         |
| **Diffusion Model** (扩散模型) | 先加噪到纯噪声，再学会逐步去噪       | 正向加噪 + 反向去噪              | 取代了 GAN 成为图像生成新王 | **Stable Diffusion、DALL-E、Midjourney** |

#### 强化学习

| 算法                      | 本质在做什么                     | 核心思想                   | 与监督学习的区别                 | 典型应用                                   |
| ------------------------- | -------------------------------- | -------------------------- | -------------------------------- | ------------------------------------------ |
| **Q-Learning / DQN**      | 智能体在环境中试错，学习最优策略 | 估计每个状态-动作对的价值  | 没有标签，只有奖励信号；边做边学 | 游戏 AI（Atari）、机器人控制               |
| **Policy Gradient / PPO** | 直接学习"在每个状态该怎么做"     | 优化策略函数使累积奖励最大 | PPO 是 ChatGPT RLHF 的核心算法   | **ChatGPT 对齐训练**、机器人、自动驾驶决策 |
| **AlphaGo / AlphaZero**   | 蒙特卡洛树搜索 + 深度网络        | 自我对弈 + 强化学习        | 超越人类的策略发现               | 围棋、国际象棋、蛋白质结构预测             |

### 三、全景关系图



```
机器学习 (从数据中学习规律)
├── 传统机器学习 (人工设计特征)
│   ├── 监督学习
│   │   ├── 分类: LR, NB, KNN, SVM, 决策树, 随机森林, XGBoost
│   │   └── 回归: 线性回归, Ridge/Lasso, SVR
│   ├── 无监督学习
│   │   ├── 聚类: K-Means, DBSCAN, 层次聚类
│   │   ├── 降维: PCA, t-SNE
│   │   └── 关联: Apriori
│   └── 半监督 / 自监督
│
└── 深度学习 (自动学习特征)
    ├── 基础网络: MLP, CNN, RNN/LSTM
    ├── Transformer 家族: BERT, GPT, ViT
    ├── 生成模型: GAN, VAE, Diffusion
    └── 强化学习: DQN, PPO, AlphaZero

数据挖掘 = 业务问题 + 数据处理 + 上述任意算法 + 结果解释
```

### 四、选型速查

| 你的数据/场景           | 首选                     |
| ----------------------- | ------------------------ |
| Excel 表格、结构化数据  | XGBoost / LightGBM       |
| 图像识别、目标检测      | CNN（ResNet、YOLO）      |
| 文本理解、搜索、问答    | BERT / GPT               |
| 时间序列预测            | LSTM / Transformer       |
| 生成图片                | Diffusion Model          |
| 生成文本                | GPT                      |
| 数据量 < 1000 条        | 传统 ML（随机森林、SVM） |
| 数据量 > 100 万条 + GPU | 深度学习                 |
| 没有标签                | 聚类 / PCA / 自监督      |
| 决策/控制/游戏          | 强化学习                 |
