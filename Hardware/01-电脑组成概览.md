# 01 - 电脑组成概览

## 一台电脑由什么组成？

把电脑想象成一个人：
- **CPU**（处理器）= 大脑，负责思考和计算
- **内存（RAM）** = 书桌，正在处理的东西放在上面，关机就清空
- **硬盘（SSD/HDD）** = 书柜，永久存放文件
- **主板** = 骨架 + 神经系统，把所有部件连在一起
- **显卡（GPU）** = 画家，专门处理图像和视频
- **电源（PSU）** = 心脏，给所有部件供电
- **散热器** = 空调，防止 CPU/GPU 过热
- **机箱** = 身体外壳，保护所有部件

## 台式机 vs 笔记本

| 对比项 | 台式机 | 笔记本 |
|--------|--------|--------|
| 体积 | 大，组件独立 | 小，高度集成 |
| 可升级性 | 几乎所有部件可换 | 通常只能换内存和硬盘 |
| 散热 | 空间大，散热好 | 空间小，散热受限 |
| 性能 | 同价位更强 | 受功耗和散热限制 |
| 维修难度 | 简单，模块化设计 | 较难，需要专用工具 |

## 核心硬件关系图（数据流向）

```
           电源 PSU
            |
     供电给所有部件
            |
   +--------+--------+
   |        |        |
  CPU --- 主板 --- GPU
   |        |        |
  散热器   内存     显示器
            |
          硬盘
```

## 关键概念

### 接口与总线
- **总线（Bus）**：数据传输的"公路"，连接各个部件
- **PCIe**：高速总线，显卡、SSD 用它连接主板
- **SATA**：较慢的接口，传统硬盘和老 SSD 用
- **USB**：外接设备的通用接口

### 兼容性
装机最重要的一件事：**确保所有部件互相兼容**
- CPU 要和主板插槽（Socket）匹配
- 内存要和主板支持的代数匹配（DDR4/DDR5）
- 电源功率要够用
- 机箱要装得下主板和显卡
