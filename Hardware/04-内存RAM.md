# 04 - 内存 RAM

## 内存是什么？

内存（RAM，Random Access Memory）是 CPU 的"工作台"。程序运行时，数据从硬盘加载到内存，CPU 从内存读取数据进行处理。

**关键特点**：断电即丢失（易失性存储）

## 为什么需要内存？

```
硬盘（慢但大）→ 加载到 → 内存（快但小）→ 送给 → CPU（最快但最小）
   ~500MB/s          ~50,000MB/s          ~1,000,000MB/s
   1-4TB              16-64GB              几十MB缓存
```

硬盘太慢，CPU 太快，内存是中间的"桥梁"。

## 核心参数

| 参数 | 含义 | 主流值 |
|------|------|--------|
| **容量** | 能存多少数据 | 16GB / 32GB / 64GB |
| **代数** | DDR 几代 | DDR4（主流）/ DDR5（新一代） |
| **频率** | 数据传输速度 | DDR4: 3200MHz / DDR5: 5600MHz |
| **时序** | 响应延迟 | CL16 / CL36（数字越小越好） |
| **电压** | 工作电压 | DDR4: 1.2V / DDR5: 1.1V |

## DDR4 vs DDR5

| 对比 | DDR4 | DDR5 |
|------|------|------|
| 频率范围 | 2133~3600MHz | 4800~8000MHz+ |
| 单条最大 | 32GB 常见 | 48GB / 64GB |
| 电压 | 1.2V | 1.1V |
| 防呆缺口位置 | 不同 | 不同（互不兼容！） |
| 价格 | 便宜 | 较贵但在降价 |

## 内存长什么样？

### 台式机内存（DIMM）
- 长条形 PCB 板，约 13.3cm 长
- 一边有防呆缺口
- 两侧可能有散热马甲（金属外壳）
- 底部一排金手指触点

### 笔记本内存（SO-DIMM）
- 比台式机短一半，约 6.7cm
- 同样有防呆缺口
- 你的 Y7000P 用的就是 SO-DIMM DDR5

## 安装内存

### 台式机
1. 打开内存插槽两端的卡扣
2. 对准防呆缺口（只有一个方向能插进去）
3. 两端同时用力按下，听到"咔嗒"声
4. 卡扣自动合上 = 安装到位

### 笔记本
1. 45度角斜插入插槽
2. 向下按压，卡扣自动锁住

### 双通道安装
- 4 个插槽时：插第 **2** 和第 **4** 槽（A2 + B2）
- 2 个插槽时：两个都插满就是双通道
- 双通道 = 两条内存同时工作，带宽翻倍

## 容量怎么选？

| 用途 | 推荐容量 |
|------|----------|
| 办公上网 | 8GB（勉强）/ 16GB |
| 游戏 | 16GB（够用）/ 32GB（推荐） |
| 视频剪辑/3D建模 | 32GB / 64GB |
| 服务器/专业工作站 | 64GB+ |

> 2025年建议起步 16GB，预算够直接 32GB

## 常见问题

**Q：内存不够会怎样？**
A：电脑会用硬盘模拟内存（虚拟内存/交换文件），速度暴降，表现为卡顿。

**Q：不同品牌/频率的内存能混用吗？**
A：物理上可以，但会降到最低频率运行，稳定性可能有问题。建议买一样的。

**Q：怎么查自己电脑的内存信息？**
A：任务管理器 → 性能 → 内存，或者用 CPU-Z 软件查看详细信息。
