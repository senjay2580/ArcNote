计算机启动，可以看成一场 三棒接力。
第一棒是固件，也就是 BIOS 或 UEFI。
它先让 CPU 从固定入口开始执行，初始化内存和基础设备，然后寻找可启动磁盘。
第二棒是引导程序，也叫 bootloader。
它找到操作系统内核，把内核加载到内存，并准备启动参数。
第三棒才轮到 内核。
内核运行以后，会建立中断、内存管理、进程管理和设备管理。
所以做 408 题时，不要把这三层混在一起。
看到上电、自检、启动项，优先想到固件。
看到加载内核、跳转内核入口，优先想到引导程序。
看到中断、页表、进程、设备，才是操作系统内核真正接管。
一句话记忆：固件找入口，引导搬内核，内核管全局。
