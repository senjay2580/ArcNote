## 一、英语二翻译题基本信息

| 项目 | 详情 |
|---|---|
| **题型** | 段落翻译（英译汉）|
| **分值** | **15 分**（占总分 10%）|
| **长度** | 约 **150 词**，1–2 个英语段落 |
| **考察能力** | 理解英语语言材料 + 准确完整通顺地译成汉语 |
| **时间建议** | 25–30 分钟 |
| **题材** | 近 10 年以社会文化、科技发展、教育健康、环境可持续为主 |

### 英语二 vs 英语一 翻译差异

| 对比项 | 英语一 | **英语二** |
|---|---|---|
| 题型 | 5 个划线长难句 | **150 词段落** |
| 长度 | 5 句约 150 词 | **150 词整段** |
| 难度 | **语法结构复杂**（专业性强） | 语法相对简单，**强调通顺** |
| 得分重点 | 每句拆分组合精确度 | **全段流畅度 + 无漏译** |

## 二、官方评分标准（研招网发布）

> **英语二翻译四档评分 · 必须背死**

| 档位 | 分数 | 标准 |
|---|---|---|
| **第四档** | **13–15** | 很好完成任务。**理解准确无误**；**表达通顺清楚**；**没有错译、漏译** |
| 第三档 | 9–12 | 基本完成任务。理解基本准确；表达比较通顺；没有**重大**错译漏译 |
| 第二档 | 5–8 | 未按要求完成。理解不够准确；表达欠通顺；有明显错译漏译 |
| 第一档 | 0–4 | 未完成任务。不能理解原文；表达不通顺；文字支离破碎 |

### 三大硬性标准

1. **准确**：不得有篡改、歪曲、遗漏、增删
2. **完整**：每句话必须是一句**完整的中文**，不能半句话断掉
3. **通顺**：符合中文表达习惯，**宁可意译也不要字字对译**

**铁律**：**宁愿通顺不漏译，也不要字对字翻译得别扭**。阅卷老师看的是"中文读起来是否像人话"，而不是"英语单词对应"。

## 三、唐静"拆分与组合翻译法"· 核心方法论

> **来源**：《考研英语拆分与组合翻译法》（新东方），考研翻译领域最有影响力的方法论
>
> **核心理念**：
> > "把艰涩的英语切换成简单的中文" — 唐静
>
> **两大步骤**：
> 1. **拆分**（Split）— 把长难句按结构信号切成小单位
> 2. **组合**（Combine）— 按中文习惯重新排列组合成通顺译文

### 3.1 四步做题法（唐静官方步骤）

| 步骤 | 操作 | 时间 |
|---|---|---|
| **Step 1 · 通读全句** | 不要抠词，先找**连接词**（and / but / because / which 等）理清逻辑关系 | 30 秒 |
| **Step 2 · 分析成分，划分意群** | 根据连接点把句子拆成**若干语法单位**（主句 / 从句 / 短语）| 30 秒 |
| **Step 3 · 选择词义** | 重点处理**多义词**，结合上下文定词义 | 5–8 分钟 |
| **Step 4 · 调整语序，得出译文** | 按中文习惯**重新排列**，必要时"词语调整 + 顺序调整" | 5–8 分钟 |

**时间分配**：前两步**总共 1 分钟**内完成，后两步才是主战场。

---

### 3.2 拆分六大标志（唐静拆分信号词）

看到以下标志立即切一刀：

| # | 标志类型 | 典型信号词 | 例子 |
|---|---|---|---|
| ① | **标点符号** | , ; : — () | 逗号分号冒号破折号括号 |
| ② | **并列连词** | and / but / or / yet / so / for / nor | `A and B` 两部分拆开 |
| ③ | **从属连词** | because / although / while / if / when / that / as | 从句开始处拆 |
| ④ | **关系代词/副词** | which / who / whom / whose / where / when | 定从 / 状从开始 |
| ⑤ | **介词短语** | of / in / at / with / for + 名词 | 长介词短语独立成块 |
| ⑥ | **分词/不定式短语** | -ing / -ed / to do | 非谓语短语独立 |

**拆分口诀**：**"标点并列从属连，关系介词非谓前"**

---

### 3.3 组合四大原则（唐静组合法则）

拆完不是终点，**重组才是得分关键**。四条铁律：

#### 原则 1 · 定语前置
英语里"名词 + 定语从句/分词短语/介词短语" → 中文里**放到名词前**，前面加"的"。

```
英文：the man who stood at the door wearing a hat
拆分：the man / who stood at the door / wearing a hat
组合：【站在门口的 / 戴着帽子的】那个男人
       ↑ 定语前置             ↑ 定语前置
```

#### 原则 2 · 状语灵活移位
英语状语经常在句末，中文习惯放句首或动词前。

```
英文：He failed in the exam because he didn't review.
直译（扣分）：他失败在考试因为他没复习。
组合（得分）：因为他没复习，所以考试没及格。（因果前置）
```

#### 原则 3 · 被动变主动
英语爱用被动，中文爱用主动或"把/被"字结构。

```
英文：The problem has been solved.
僵硬译法：这个问题已经被解决了。
自然译法：问题（已经）解决了。 / 人们已经解决了这个问题。
```

#### 原则 4 · 长句拆短句
英语一句 40 词常见，中文读不通。**遇到超过 20 字的中文句子，就断开**。

```
英文：It is believed that education is the key to success, 
     which has been proven by countless examples throughout 
     human history.

字字译法（扣分）：人们相信教育是成功的关键，它已经被贯穿人类历史
的无数例子所证明。
（一句话 33 字，读起来很累）

拆分组合（得分）：人们相信，教育是成功的关键。纵观人类历史，
无数事例都证明了这一点。
（拆成两句，读起来流畅）
```

## 四、七大考点 + 对应翻译技巧

> **来源**：研招网官方考点归纳 + 唐静讲义

### 考点 1 · 定语从句

**处理三法**：
- **短从句**（≤ 6 词）→ **前置翻译**
  - `the book that I bought` → 我买的那本书
- **长从句**（> 6 词）→ **后置翻译**，用"这 / 该 / 它"引出
  - `The method, which was invented 50 years ago, is still useful.`
    → 这种方法是 50 年前发明的，**至今仍然有用**。
- **非限定定从** → 后置 + 补充说明

---

### 考点 2 · 状语从句

**处理两法**：
- **原因/条件/让步从句** → **前置**（符合中文习惯）
  - `He succeeded because he worked hard.`
    → **因为他努力工作**，他才成功。
- **时间/方式/结果从句** → 保持**原位**
  - `He finished the task when the sun set.`
    → 他在太阳落山时完成了任务。

---

### 考点 3 · 被动语态

**处理四法**：
- **保留"被"字**：强调被动行为时
  - `He was arrested.` → 他**被**逮捕了。
- **改为主动**：有明显动作发出者时
  - `The book was written by him.` → 这本书**是他写的**。
- **无主句**：没有明确主语时
  - `It is said that ___.` → **据说** ___。
- **"人们 / 大家 / 有人"** 补主语
  - `It is believed that ___.` → **人们相信** ___。

**高频被动固定译法**：
```
It is said / reported that ___       据说 / 据报道
It is believed / thought that ___    人们相信 / 认为
It is well known that ___            众所周知
It must be admitted that ___         必须承认
It cannot be denied that ___         不可否认
It should be emphasized that ___     应当强调
```

---

### 考点 4 · 非谓语动词

**-ing / -ed / to do 做状语**：
- 开头 → 译成"**在...时 / 由于 / 为了**"
  - `Walking in the park, I met him.` → **在公园散步时**，我遇到了他。
- 中间做后置定语 → 前置加"的"
  - `the book written by him` → **他写的**那本书

---

### 考点 5 · 多义词定词义

唐静强调：**80% 的翻译错误来自多义词选错意思**。

**处理口诀**：**"词不离句，句不离文"** — 根据段落大意反推词义。

**英语二高频多义词 TOP 15**：

| 词 | 常见意 | 考研易考意 |
|---|---|---|
| work | 工作 | **作品 / 起作用 / 运转** |
| subject | 主题 / 学科 | **使服从 / 受制于** |
| address | 地址 | **演讲 / 致辞 / 解决** |
| account | 账户 | **解释 / 说明 / 重要性** |
| appreciate | 欣赏 | **理解 / 意识到** |
| develop | 发展 | **患上（疾病）/ 养成（习惯）** |
| observe | 观察 | **遵守 / 评论** |
| claim | 声称 | **索取 / 夺去（生命）** |
| succeed | 成功 | **继任 / 继承** |
| present | 现在 / 礼物 | **呈现 / 介绍 / 出席的** |
| issue | 问题 | **发行 / 颁发** |
| matter | 物质 | **重要（v.）** |
| challenge | 挑战 | **质疑** |
| capture | 捕获 | **吸引（注意）** |
| maintain | 保持 | **主张 / 维持观点** |

---

### 考点 6 · 代词回指

英语代词（it / they / this / these）频繁出现，**必须翻译出所指对象**才能得分。

```
英文：Some people believe that technology is the answer. 
     However, this view is not always correct.

错误翻译：一些人认为技术是答案。然而，**这个观点**并不总是正确的。
（this 是什么？阅卷人看不到）

正确翻译：一些人认为技术是答案。然而，**这种认为技术是答案的观点**
并不总是正确的。
（明确把 this 回指到前文）
```

---

### 考点 7 · 插入语

英语喜欢在主谓之间插入解释（`, however, / , for example, / , of course,`），**中文翻译时提到句首或合并**。

```
英文：The method, however, is not always effective.
僵硬译法：这种方法，然而，并不总是有效的。
自然译法：然而，这种方法并不总是有效的。
```

## 五、做题避坑 · 5 大翻译常见雷区

| # | 雷区 | 症状 | 避开方法 |
|---|---|---|---|
| ① | **字字对译** | 中文读起来像外语 | 先理解再翻译，**宁可多字少字也要通顺** |
| ② | **漏译定语** | which 从句忘翻了 | 拆分时用红笔标出**每一个从句** |
| ③ | **多义词错选** | 把 work 翻成"工作"（实际是"作品"）| 回看上下文，尤其是前后名词 |
| ④ | **被动生硬** | 满篇"被"字 | **70% 的被动句要改写为主动或无主句** |
| ⑤ | **长句不断** | 一句 40 字读不通 | **超过 20 字立即断句**，加句号重起 |

## 六、实战演示 · 一句话的完整拆分组合过程

**原句**（2018 英语二真题第一句改编）：
> A fifth-grader gets homework assignments on Saturday mornings from his elder brother, **who is doing well in college but reluctant to help his younger siblings**, **because he believes it is a waste of time**.

### Step 1 · 通读 + 找连接词
- `who`（关系代词）· `but`（并列）· `because`（从属）

### Step 2 · 拆分意群

```
A fifth-grader gets homework assignments on Saturday mornings from his elder brother,
│
├── who is doing well in college
│   │
│   but (并列)
│   │
├── (who is) reluctant to help his younger siblings,
│
└── because he believes it is a waste of time.
```

### Step 3 · 逐块翻译 + 选词义

| 英文块 | 中文块 | 备注 |
|---|---|---|
| A fifth-grader gets homework assignments on Saturday mornings from his elder brother | 一位五年级学生每周六早上从他哥哥那里拿到家庭作业 | `assignments` = 作业任务 |
| who is doing well in college | 他哥哥在大学里成绩优异 | `doing well` = 表现好 |
| but reluctant to help his younger siblings | 但不愿意帮助弟弟妹妹 | `reluctant to` = 不愿意 |
| because he believes it is a waste of time | 因为他认为这是浪费时间 | 状从前置 |

### Step 4 · 组合 + 调整

**直译**（扣分）：
> 一位五年级学生每周六早上从他哥哥那里拿到家庭作业，他哥哥在大学里成绩优异但不愿意帮助弟弟妹妹，因为他认为这是浪费时间。
（一句 52 字，读起来很累）

**高分组合**（拆成 2–3 句）：
> 每周六早上，一位五年级的学生都会从哥哥那里拿到家庭作业。**哥哥在大学里成绩优异，但并不愿意帮助弟弟妹妹**，因为他觉得这是浪费时间。

**为什么得分**：
- ✅ 一句话拆成两句，符合中文习惯
- ✅ 状语"每周六早上"前置
- ✅ 定语从句单独成句，补代词"哥哥"
- ✅ 并列但 = 但并不
- ✅ `it is a waste of time` → "这是浪费时间"，不做字面翻译成"它是时间的浪费"

## 十、高分翻译自检清单

**做完每一篇翻译后，用下面 8 条自检**：

```
□ 1. 中文读起来像人话，不是机器翻译腔
□ 2. 所有定语从句都翻译到位（没有漏译 which 引导句）
□ 3. 被动句 70% 以上改为主动或无主句
□ 4. 超过 20 字的中文自动断句
□ 5. 多义词都检查过上下文（不是查词典第一个意思）
□ 6. 代词 it / they / this 都明确回指对象
□ 7. 没有漏译任何一个句子（对照原文逐句勾）
□ 8. 没有增译自己理解的内容（翻译不是创作）
```

**通过 8 条 → 第四档（13-15 分）**
**通过 6 条 → 第三档（9-12 分）**
**低于 6 条 → 继续用拆分组合法训练**

## 十一、"翻译 = 英语二回血题" 的认知

> **核心认知**：英语二翻译是**性价比最高的提分题**。
>
> - **阅读**：错 1 道 -2 分，而且题目难度大
> - **作文**：上 17 分极难
> - **翻译**：**稳拿 13 分完全可能**，但大多数人只拿到 9 分，这中间有 **4 分** 白送
>
> **回血策略**：用 14 天死磕唐静方法论，可以把翻译从"随缘 8 分"提升到"稳定 13 分"，相当于**总分 +4**。
