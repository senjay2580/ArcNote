# 408 统考全模块考点分类

> 依据近 10 年真题题型分布（单选 40 × 2 分 + 综合 7 × 共 70 分）与王道/天勤模拟题出题方向整理
> 分值占比：数据结构 45 · 计算机组成原理 45 · 操作系统 35 · 计算机网络 25

---

## 一、数据结构（45 分）

### 1. 绪论与算法分析
- 数据结构三要素（逻辑结构 / 存储结构 / 运算）
- 算法时间复杂度计算（递推式、主定理、嵌套循环）
- 空间复杂度分析
- **高频题型**：给代码段求时间复杂度（选择必考 1 题）

### 2. 线性表
- 顺序表：插入/删除均摊复杂度、动态扩容
- 单链表 / 双链表 / 循环链表 / 静态链表
- **高频算法设计**：就地逆置、删除重复元素、两链表合并、寻找公共结点、中间结点、倒数第 k 个结点
- 顺序表 vs 链表对比

### 3. 栈、队列与数组
- 栈的应用：括号匹配、表达式求值（中缀→后缀→计算）、递归转非递归
- 队列：循环队列（判空判满三种方式）、双端队列合法出队序列
- 共享栈、链式队列
- 特殊矩阵压缩存储（对称、三角、三对角、稀疏矩阵）
- **高频**：卡特兰数求合法出栈序列数

### 4. 串（考查较少）
- KMP 算法 next / nextval 数组手算
- 模式匹配复杂度

### 5. 树与二叉树（重点）
- 二叉树性质（结点数、叶子数、高度关系）
- 存储结构（顺序 / 链式）
- 遍历：先/中/后/层序，已知两种序列还原二叉树
- **线索二叉树**：中序/先序线索化、寻找前驱后继
- 树、森林与二叉树的相互转换
- 哈夫曼树：构造、WPL 计算、哈夫曼编码
- 并查集（近年新增）
- **算法题高频**：判断满/完全二叉树、求高度、求宽度、WPL、相似判断、叶子链表、非递归遍历

### 6. 图（重点 + 算法题常客）
- 存储：邻接矩阵、邻接表、十字链表、邻接多重表
- 遍历：DFS / BFS（含生成树 / 生成森林）
- 最小生成树：Prim（稠密图）、Kruskal（稀疏图）手算过程
- 最短路径：Dijkstra（单源，不能有负权）、Floyd（多源）
- 拓扑排序 / 逆拓扑 / AOV 网
- 关键路径：AOE 网，ve/vl/e/l 四量计算
- **综合题高频**：邻接表存储下的 BFS/DFS 算法、判断连通性、判断是否为树

### 7. 查找
- 顺序查找、折半查找（判定树、ASL 成功/失败）
- 分块查找
- **B 树 / B+ 树**：插入删除过程、高度计算、与 B+ 差异
- 散列表：处理冲突（开放定址、拉链）、ASL 计算、装填因子
- 红黑树基本性质（近年新增）

### 8. 排序（重点）
- 插入类：直接插入、折半插入、希尔
- 交换类：冒泡、快速排序（划分过程、最坏/最好情况）
- 选择类：简单选择、堆排序（建堆/调整/删除）
- 归并排序（二路归并过程）
- 基数排序
- **必背对比表**：时间/空间复杂度、稳定性、是否适用链表
- **高频**：给出中间状态判断用的什么排序、第 k 趟结果

## 二、计算机组成原理（45 分）

### 1. 计算机系统概述
- 冯诺依曼结构、现代计算机组织
- 性能指标：CPI、MIPS、MFLOPS、Amdahl 定律
- 机器字长、指令字长、存储字长

### 2. 数据的表示与运算（重点 + 综合题常客）
- 进制转换（二/八/十/十六）
- 定点数：原码、反码、补码、移码（表示范围、转换）
- **溢出判断**：双符号位、单符号位异或
- 补码加减法运算、乘法（Booth 算法）、除法（恢复/不恢复余数）
- IEEE 754 单/双精度浮点数：规格化、特殊值、表示范围
- **浮点数运算**：对阶、尾数加减、规格化、舍入、判溢出
- C 语言数据类型强制转换中的机器数变化
- 电路层：全加器、串/并行进位加法器、ALU

### 3. 存储系统（重点）
- 分类：SRAM / DRAM（刷新方式）、ROM、Flash
- 主存扩展：位扩展、字扩展、字位同时扩展
- 多模块存储器（单体多字、多体交叉）
- **Cache**：
  - 映射方式（直接、全相联、组相联）地址划分
  - 替换算法（LRU、FIFO、随机）
  - 写策略（全写法、写回法、写分配 / 非写分配）
  - 命中率、平均访问时间计算
- **虚拟存储器**：
  - 页式、段式、段页式
  - TLB、页表、Cache 三级协同访问过程
  - 缺页处理
- **高频综合题**：Cache + 主存 + 虚存三级访问地址变换

### 4. 指令系统
- 指令格式、地址码数目
- 寻址方式（立即、直接、间接、寄存器、寄存器间接、相对、基址、变址、堆栈）
- **CISC vs RISC** 对比
- 程序的机器级表示（近年新增 AT&T 汇编语法识别、过程调用栈帧、选择/循环/数组访问）
- x86 汇编读代码填结果

### 5. 中央处理器（重点 + 必出综合题）
- CPU 功能、基本结构（运算器 + 控制器）
- 数据通路：单总线、多总线
- **指令执行过程**：取指 → 间址 → 执行 → 中断周期的微操作序列
- 控制器：硬布线、微程序控制
- 微指令格式（水平型、垂直型）、微地址形成方式
- **指令流水线**：
  - 基本概念、性能指标（吞吐率、加速比、效率）
  - 三种相关（结构、数据、控制）及解决
  - 超标量、超流水、动态调度
- 多处理器系统（SIMD / MIMD / 向量处理器 / 共享存储）
- **综合题高频**：给指令系统画数据通路 + 写微操作序列

### 6. 总线
- 分类：片内、系统、通信总线
- 总线仲裁（链式查询、计数器定时查询、独立请求）
- 总线定时（同步、异步、半同步、分离式）
- 带宽计算

### 7. 输入输出系统
- I/O 接口结构、端口编址（统一/独立）
- **五种 I/O 控制方式**：程序查询、程序中断、DMA、通道、I/O 处理机
- 中断处理过程、多重中断、中断屏蔽字
- DMA 工作过程、与中断区别
- **高频**：中断响应时间、DMA 传送字数/时间计算

## 三、操作系统（35 分）

### 1. 操作系统概述
- 特征（并发、共享、虚拟、异步）
- 运行机制：用户态/核心态、特权指令、中断与异常、系统调用
- 体系结构：大内核、微内核
- **高频选择**：哪些操作需陷入内核态

### 2. 进程与线程（重点 + 必出综合题）
- 进程状态转换（五状态模型）、PCB
- 线程（用户级 / 内核级 / 混合）
- **调度算法**：FCFS、SJF/SRTN、优先级、RR、多级反馈队列、HRRN
- 周转时间 / 带权周转时间 / 等待时间计算
- **进程同步（必考综合题）**：
  - PV 操作、信号量机制
  - 经典问题：生产者-消费者、读者-写者、哲学家进餐、理发师
  - 管程（Hoare / Mesa）
- **死锁**：
  - 四必要条件
  - 银行家算法
  - 死锁检测与解除
  - 资源分配图化简

### 3. 内存管理（重点）
- 连续分配（固定分区、动态分区 + 首次/最佳/最坏适应）
- 分页、分段、段页式（地址结构、地址变换、TLB）
- **虚拟内存**：
  - 请求分页系统
  - 页面置换算法（OPT、FIFO、LRU、CLOCK、改进 CLOCK）
  - Belady 异常
  - 工作集、抖动
  - 驻留集大小、页面分配策略（固定/可变 + 局部/全局）
- **高频**：多级页表级数计算、有效访存时间、缺页率与命中率

### 4. 文件管理
- 文件逻辑结构（无结构、有结构 + 顺序/索引/索引顺序）
- **物理结构**：连续、链接（隐式/显式/FAT）、索引（单/多级/混合）
- 目录结构（单级、两级、树形、无环图）、FCB、索引结点
- **文件系统**：Unix 混合索引、目录实现
- 文件共享（硬链接 / 软链接）
- 文件保护（口令、加密、访问控制表）
- 空闲空间管理（位示图、空闲链表、成组链接）
- 磁盘调度：FCFS、SSTF、SCAN、CSCAN、LOOK、CLOOK
- 磁盘访问时间（寻道 + 旋转 + 传输）
- RAID 各级特性

### 5. 输入输出管理
- I/O 设备分类、I/O 控制方式（与组原重叠但侧重软件）
- I/O 软件层次（用户层、设备独立性、设备驱动、中断处理）
- 假脱机 SPOOLing
- 缓冲区（单 / 双 / 循环 / 缓冲池）
- 设备分配与回收

## 四、计算机网络（25 分）

### 1. 计算机网络体系结构
- 分类、性能指标（带宽、吞吐量、时延 = 发送+传播+处理+排队）
- 时延带宽积、往返时延 RTT、利用率
- OSI 七层、TCP/IP 四层、五层参考模型对比
- 各层 PDU 名称、分用复用

### 2. 物理层
- 数据通信基础（信道、奈奎斯特、香农公式）
- 编码（不归零、曼彻斯特、差分曼彻斯特、4B/5B）
- 调制（ASK、FSK、PSK）
- 传输介质（双绞线、同轴、光纤）
- 物理层设备（中继器、集线器）

### 3. 数据链路层（重点）
- 三基本问题：封装成帧、透明传输（字符填充 / 零比特填充）、差错控制
- **差错控制**：奇偶校验、CRC（手算）、海明码（纠错位数、编码）
- **流量控制与可靠传输**：停止等待、GBN、SR
  - 窗口大小与序号位数关系
  - 信道利用率计算
- **介质访问控制**：
  - 静态（FDM、TDM、WDM、CDMA）
  - 动态随机（ALOHA、CSMA、CSMA/CD、CSMA/CA）
  - 轮询（令牌环）
- **以太网**：MAC 帧格式、最小/最大帧长、CSMA/CD 争用期、二进制退避
- PPP 协议、HDLC
- 局域网、VLAN
- 链路层设备：网桥、交换机（自学习算法）

### 4. 网络层（重点 + 综合题常客）
- **IPv4**：
  - 首部格式
  - 分类 IP / 子网划分 / CIDR
  - 子网掩码、路由聚合
  - **高频**：IP 地址规划、可分配主机数
- ARP、DHCP、ICMP（差错报告 + 询问报文）
- **路由算法**：距离向量（RIP）、链路状态（OSPF）、路径向量（BGP）
  - 区别与适用场景
  - RIP 更新过程、最多 15 跳
- IP 组播、IGMP
- 移动 IP
- IPv6 地址格式、与 v4 过渡
- 路由器结构、转发 vs 路由
- **综合题高频**：子网划分 + 路由表构造 + 转发过程

### 5. 传输层（重点）
- UDP：简单首部、无连接、面向报文
- **TCP（必考）**：
  - 首部格式、序号确认号
  - 三次握手、四次挥手（状态机）
  - 可靠传输：累积确认、选择确认 SACK
  - **流量控制**：滑动窗口、零窗口探测
  - **拥塞控制**：慢开始、拥塞避免、快重传、快恢复（画图）
  - 定时器、超时重传 RTO 计算（RTT 平滑）
- **高频**：给 TCP 报文段分析序号/确认号/数据字节范围、窗口大小变化

### 6. 应用层
- DNS：域名结构、递归 / 迭代查询、资源记录
- FTP：控制连接 + 数据连接、主动 / 被动模式
- 电子邮件：SMTP / POP3 / IMAP / MIME
- **HTTP**：请求 / 响应报文格式、方法、状态码、持久连接、Cookie
- HTTPS 基本概念（近年新增）
- DHCP、SNMP、Telnet

## 五、跨科目综合考点（常出现在压轴综合题）

| 跨科目方向 | 涉及章节 |
|------------|----------|
| Cache + 虚存 + 页表 + TLB 联合访问 | 组原存储 + OS 内存 |
| 进程调度 + 中断响应 + CPU 周期 | 组原 CPU + OS 进程 |
| 文件系统 + 磁盘调度 + 数据结构索引 | OS 文件 + DS 树 |
| TCP 滑动窗口 + 链路层窗口差异 | 网络传输层 + 链路层 |
| C 代码翻译汇编 + 栈帧变化 | 组原指令系统 + 数据结构栈 |

## 六、备考建议（基于近年出题趋势）

1. **算法题必刷**：DS 图遍历、链表操作、二叉树递归/非递归
2. **综合大题训练**：CPU 数据通路、Cache 地址变换、PV 同步、子网划分 + TCP
3. **新增考点关注**：红黑树、并查集、x86 汇编、HTTPS、IPv6、多处理器
4. **易错陷阱**：补码溢出判断、浮点规格化、页表级数、TCP 序号、子网零号网络
