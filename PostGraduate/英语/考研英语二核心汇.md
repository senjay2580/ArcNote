## 五、各题型核心考点速览

### 5.1 完形填空（10 分）

**出题规律**：
- 4 选 1，约 350 词
- **70% 考词汇**（实词辨析 + 搭配），**30% 考逻辑**（连接词 + 语篇关系）
- 难点：**近义词辨析**（如 `influence / affect / impact`）

**高频考点**：
- 动词搭配（接名词还是介词）
- 连词：however / moreover / therefore / nevertheless
- 限定词：all / some / few / most

**核心策略**：
1. **先读首句和末句**（常含主旨）
2. 整体通读建立语境
3. **空与空之间有呼应** — 前文提到的词很可能是后文空的答案
4. 用**排除法**：明显错误的 2 个选项先划掉，再从剩 2 个选

---

### 5.2 阅读理解 Part A · Text 1–4（40 分，**最核心**）

**出题规律**：
- 4 篇文章，每篇约 400 词 × 5 题
- 每篇 2 分 / 题
- 题型分布（固定比例）：
  - **细节题** 50% → 回原文定位
  - **主旨题** 20% → 看首尾段
  - **推断题** 15% → 基于事实合理延伸
  - **词义题** 10% → 上下文推测
  - **态度题** 5% → 关注情感形容词

**高分策略**（唐迟阅读体系）：

1. **"干扰项特征"识别**
   - 无中生有：原文没提
   - 偷换概念：改动关键限定词
   - 以偏概全：个别代替整体
   - 强加因果：硬凑逻辑
   - 正反混淆：否定词漏掉

2. **定位原则**：题目里的**专有名词 / 时间 / 数字 / 大写字母**直接回原文找

3. **态度词清单**：
   - 积极：`optimistic / supportive / positive / enthusiastic`
   - 消极：`skeptical / critical / concerned / doubtful`
   - 中立：`objective / neutral / indifferent`
   - **注意**：`impartial / detached` 多为干扰项，英语二阅读很少出现真正的"中立"

---

### 5.3 新题型 Part B（10 分）

**两种题型**（每年二选一）：
- **多项对应**（5 选 5 / 7 选 5）：给段落找对应关系
- **小标题匹配**：给段落找标题

**出题规律**：
- **偶数年多考匹配，奇数年多考多项对应**（近年已打破规律，不可押题）
- 难度：**低于阅读 Text**，但需通读全文

**高分策略**：
1. **先读选项再看文章**
2. **首句末句 + 段内关键词**是判断依据
3. 用**排除法**：先定 3 个最确信的，再猜 2 个不确信的
4. **重复词汇 / 同义替换** 是最强信号

---

### 5.4 翻译（15 分）

> **核心方法论**：唐静"拆分与组合翻译法"（详见 [考研英语二翻译.md](考研英语二翻译.md)）

**简略步骤**：
1. **通读全句找连接词**（30 秒）
2. **拆分意群**（30 秒）
3. **选词义**（5-8 分钟）
4. **调整语序得出译文**（5-8 分钟）

**得分铁律**：
- **宁可意译不要字字对译**
- **被动句 70% 改主动 / 无主**
- **超过 20 字中文自动断句**
- **定语从句短前置、长后置**

---

### 5.5 写作（25 分）

> **核心方法论**：石雷鹏 30 功能句法 + 王江涛模板（详见 [考研英语二作文模板.md](作文/考研英语二作文模板.md)）

**简略策略**：
- **小作文 10 分**：三段式 · 目的 / 正文 / 期待回复 · 100 词 · 邮件体
- **大作文 15 分**：三段式 · 数据 / 原因 / 评论 · 150-200 词 · 图表作文
- **阅卷秘密**：阅卷 30-60 秒，**印象分定档 + 亮点句型拉分**
- **必须出现 2-3 个亮点句**：倒装 / 虚拟 / 非谓语 / 强调 / 定从

## 六、高频考点通用规律

### 6.1 阅读 + 完形 通用 · 逻辑连接词

| 类别 | 词 | 在阅读中的作用 |
|---|---|---|
| **转折** | however / yet / nevertheless / nonetheless / by contrast | 题目答案常在转折词后 |
| **因果** | because / since / as / therefore / hence / thus | 推断题考因果 |
| **并列递进** | moreover / furthermore / in addition / besides | 细节题考递进 |
| **举例** | for instance / for example / such as / a case in point | 举例前后是考点 |
| **让步** | although / though / even if / despite | 让步之后常是作者真正观点 |

### 6.2 翻译 + 阅读 通用 · 难句处理口诀

> **"标点并列从属连，关系介词非谓前"**

看到这些信号词立即切一刀：
1. 标点（, ; : — ）
2. 并列连词（and / but / or）
3. 从属连词（because / although）
4. 关系词（which / who / that）
5. 介词短语（of / in / with 长短语）
6. 非谓语（-ing / -ed / to do）

### 6.3 作文 + 翻译 通用 · 同义替换 10 组

| 烂大街 | 高分替换 |
|---|---|
| many | a multitude of / numerous / a host of |
| people | individuals / citizens / the public |
| important | crucial / vital / indispensable |
| good | favorable / beneficial / rewarding |
| bad | detrimental / adverse / undesirable |
| think | deem / maintain / hold the view that |
| show | reveal / illustrate / demonstrate |
| because | owing to / attributed to / on account of |
| so | hence / therefore / consequently |
| more and more | a growing number of |

## 十、易错 / 易忽视清单（考前必看）

1. ❌ 只背单词不做真题 → 题感永远上不来
2. ❌ 阅读只看文章不看错题 → 同一个坑踩两次
3. ❌ 翻译用机器翻译验证 → 养成依赖，考场崩盘
4. ❌ 作文只背 1 套模板 → 题目一变就全废
5. ❌ 完形用排除法乱猜 → 正确率卡在 50%
6. ❌ 忽视新题型 → 10 分白丢
7. ❌ 考前才开始练手写 → 字数和卷面控制不住
8. ❌ 只做近 5 年真题 → 题型演变规律看不到
9. ❌ 考前背长难句 PDF → 来不及消化
10. ❌ 把英语一真题当英语二做 → 难度风格完全不同
