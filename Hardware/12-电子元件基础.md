# 12 - 电子元件基础

## 为什么要认识电子元件？

看懂主板上密密麻麻的小零件，能帮你：
- 判断哪个元件烧了/鼓包了
- 理解为什么某个部位会出问题
- 和维修师傅/客服沟通时不被忽悠

## 主板上的核心元件

### 1. 电容（Capacitor）

```
外观：小圆柱体 或 小方块

固态电容（好）        电解电容（一般）
  ┌──┐                  ┌──┐
  │  │ 没有十字          │✕│ 顶部有十字刻痕
  │  │                   │  │
  └──┘                   └──┘
```

**作用**：存储电荷、滤波、稳压
**常见问题**：
- **鼓包/漏液**：电解电容顶部鼓起来了 → 需要更换
- 这是老电脑主板最常见的故障之一

### 2. 电阻（Resistor）

```
外观：主板上非常小的长方形贴片（贴片电阻）
     [████]  通常是黑色或棕色的小方块
```

**作用**：限制电流、分压
**在主板上**：到处都是，密密麻麻的小颗粒

### 3. 电感（Inductor）

```
外观：方形带线圈的元件，CPU供电区域最密集

  ┌──────┐
  │ ≋≋≋≋ │  ← 里面有线圈
  └──────┘
```

**作用**：滤波、储能（VRM 供电模块的核心组件）
**常见问题**：电感啸叫（高频嗡嗡声），一般不影响使用

### 4. MOSFET（场效应管）

```
外观：CPU供电区域的黑色小方块，通常有散热片覆盖

  ═══════  ← 供电散热片
  ■ ■ ■ ■  ← MOSFET
  [CPU插槽]
```

**作用**：VRM 供电的核心开关元件，控制给 CPU 的电压
**常见问题**：过热烧毁（摸着烫手 = 不正常）

### 5. 晶振（Crystal Oscillator）

```
外观：银色小金属罐，通常标注频率

  ╭───╮
  │32.768│  ← 频率值
  ╰───╯
```

**作用**：提供精确的时钟频率信号（像心跳一样给各部件打节拍）

### 6. BIOS 芯片

```
外观：方形芯片，通常有品牌标识

  ┌──────────┐
  │ WINBOND  │  或 MXIC 等品牌
  │ 25Q128  │
  └──────────┘
```

**作用**：存储 BIOS/UEFI 固件
**维修相关**：BIOS 损坏可以用编程器重新刷写

### 7. 纽扣电池（CMOS 电池）

```
外观：一个银色圆形纽扣电池（CR2032）

    ╭────╮
    │CR  │
    │2032│
    ╰────╯
```

**作用**：给 BIOS 供电，保存时间和 BIOS 设置
**小技巧**：拔掉电池等 30 秒再装回 = 重置 BIOS 设置（忘密码时有用）

### 8. 接口芯片

| 芯片 | 作用 |
|------|------|
| **网卡芯片**（Realtek/Intel） | 提供有线网络 |
| **声卡芯片**（Realtek ALC） | 提供音频输入输出 |
| **USB 控制器** | 管理 USB 接口 |
| **雷电控制器** | Thunderbolt 接口 |

## 常见的"坏了"长什么样

### 电容鼓包
```
正常          鼓包          漏液
┌──┐         ┌──┐         ┌──┐
│  │         │⌒│ ←顶部鼓起 │  │ ←周围有胶状物
│  │         │  │         │  │
└──┘         └──┘         └──┘
```

### 烧焦痕迹
- 元件表面变黑/变色
- 周围 PCB（电路板）有焦黄色
- 可能有烧焦气味

### 焊点虚焊/裂纹
- 焊接点看起来粗糙、不光滑
- 元件松动
- 用放大镜或显微镜能看到裂纹

## PCB（印刷电路板）基础

主板本身就是一块多层 PCB：
- **表面**：绿色/黑色的阻焊层（防止短路）
- **内部**：多层铜走线（4-8层常见）
- **金手指**：接口上的金色触点，镀金防氧化
- **过孔**：连接不同层的小洞

## 维修时的注意事项

1. **断电！** 任何维修操作前必须断电拔线
2. **防静电**：戴防静电手环或频繁触碰金属放电
3. **不要盲目拆**：先拍照记录原样
4. **烙铁焊接**：需要专门学习和练习，新手不建议自己焊
5. **万用表**：学会用万用表测量电压/电阻/通断，是维修的基本技能
