# Reitti 项目概述

## 项目简介

**Reitti** 是一个功能完善的个人位置追踪与分析应用，名称来源于芬兰语，意为"路线"或"路径"。这是一个自托管的隐私优先解决方案，帮助用户理解自己的移动模式和重要地点。

## 核心价值

1. **隐私优先**: 所有位置数据存储在用户自己的服务器上，永不外泄
2. **自托管**: 完全部署在用户自己的基础设施上
3. **多数据源**: 支持多种数据导入格式和实时追踪
4. **智能分析**: 自动检测访问地点、行程和交通方式

## 数据处理流程

```
数据输入 → 队列处理 → 分析检测 → 存储索引 → 可视化展示

1. 数据摄入
   ├── 文件上传 (GPX, Google Takeout, GeoJSON)
   ├── 实时追踪 (OwnTracks, GPSLogger, Overland)
   └── REST API

2. 队列处理
   ├── RabbitMQ 异步处理
   ├── 批量处理优化
   └── 死信队列处理

3. 分析检测
   ├── 访问检测 (停留点识别)
   ├── 行程分析 (路径计算)
   └── 交通方式推断 (速度分析)

4. 存储索引
   ├── PostGIS 空间索引
   ├── 时间范围索引
   └── Redis 缓存

5. 可视化展示
   ├── 交互式地图
   ├── 时间线视图
   └── 统计图表
```

## 主要功能模块

1. **位置追踪**: 实时和批量位置数据收集
2. **访问检测**: 自动识别停留地点
3. **行程分析**: 计算路径、距离、时长
4. **交通方式**: 智能推断步行、骑行、驾车等
5. **地点管理**: 重要地点命名和分类
6. **照片集成**: Immich 照片服务器集成
7. **多用户**: 支持多用户和数据共享
8. **国际化**: 支持多语言 (英、芬、德、法、中等)
