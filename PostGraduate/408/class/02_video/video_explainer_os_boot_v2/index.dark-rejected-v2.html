EP.01 — BOOT PROCESS
计算机启动，可以看成一场三棒接力。
第一棒是固件，也就是 BIOS 或 UEFI。
它先让 CPU 从固定入口开始执行，初始化内存和基础设备，再按启动顺序找可启动磁盘。
第二棒是引导程序，也叫 bootloader。
它不管进程、不管文件系统全局——只找内核、装入内存、跳转入口。
第三棒才轮到 内核。
内核运行后，建立中断 · 内存 · 进程 · 设备四套子系统。
做 408 题时，不要把这三层混在一起。
看到 上电、自检、启动项 → 想到固件。
看到 加载内核、跳转入口 → 想到引导程序。
看到 中断、页表、进程、设备 → 内核接管。
一句话记忆：固件找入口，引导搬内核，内核管全局。
