## 项目技术栈速览

```
后端: Java 24 + Spring Boot 3.5 + PostgreSQL/PostGIS + RabbitMQ + Redis
前端: Thymeleaf + HTMX + Leaflet.js
部署: Docker + Docker Compose
测试: JUnit 5 + Testcontainers + Playwright
```

## 核心功能

- 📍 位置追踪与分析
- 🚗 交通方式自动识别
- 📸 照片集成 (Immich)
- 👥 多用户支持
- 🔐 多种认证方式
- 🌍 多语言国际化
- 🏠 完全自托管
