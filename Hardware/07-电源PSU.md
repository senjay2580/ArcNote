# 07 - 电源 PSU

## 电源是什么？

电源（PSU，Power Supply Unit）把墙壁插座的 220V 交流电转换成电脑各部件需要的 12V / 5V / 3.3V 直流电。

> 电源是最容易被忽视但最重要的部件之一。劣质电源可能烧毁其他硬件！

## 核心参数

| 参数 | 含义 | 建议 |
|------|------|------|
| **额定功率** | 能稳定输出的最大功率 | 根据配置选，留 100-150W 余量 |
| **转换效率** | 多少电能被有效利用 | 80 Plus 认证越高越好 |
| **模组类型** | 线材是否可拆卸 | 全模组最方便 |
| **尺寸** | ATX / SFX | 看机箱支持哪种 |

## 80 Plus 认证等级

```
80 Plus 白牌    ████████░░  80% 效率
80 Plus 铜牌    █████████░  85% 效率    ← 性价比之选
80 Plus 金牌    █████████▓  90% 效率    ← 推荐
80 Plus 铂金    ██████████  92% 效率
80 Plus 钛金    ██████████  94% 效率
```

> 效率越高 = 越省电 + 发热越少 + 通常做工越好

## 模组类型

| 类型 | 说明 | 优缺点 |
|------|------|--------|
| **非模组** | 所有线材焊死在电源上 | 便宜，但多余的线堆在机箱里 |
| **半模组** | 主板供电线焊死，其他可拆 | 折中方案 |
| **全模组** | 所有线材可拆卸 | 整洁，但贵。只接需要的线 |

> **警告**：不同品牌的模组线不能混用！接口看着一样但线序可能不同，混用会烧硬件！

## 电源线材与接口

| 线材 | 用途 | 说明 |
|------|------|------|
| **24pin ATX** | 主板主供电 | 最大的一个接头 |
| **4+4pin CPU** | CPU 辅助供电 | 插主板左上角 |
| **6+2pin PCIe** | 显卡供电 | 高端显卡需要 1-3 个 |
| **12VHPWR** | 新一代显卡供电 | RTX 40/50 系列用 |
| **SATA 电源** | 硬盘/风扇供电 | 扁平 L 形接头 |
| **大4pin (Molex)** | 老设备供电 | 基本淘汰 |

## 功率怎么选？

### 简单估算法
```
CPU TDP + 显卡 TGP + 100W（其他部件）+ 100W（余量）= 推荐电源功率
```

### 常见配置推荐
| 配置 | 推荐电源 |
|------|----------|
| i5 + 无独显 | 350-450W |
| i5 + RTX 4060 | 550W |
| i7 + RTX 4070 | 650W |
| i7 + RTX 4080 | 850W |
| i9 + RTX 4090 | 1000W+ |

## 安装电源

1. 电源风扇朝下（如果机箱底部有进风口）
2. 从机箱后方推入电源仓
3. 用4颗螺丝固定
4. 依次接线：
   - 24pin → 主板
   - 8pin CPU → 主板左上
   - PCIe → 显卡
   - SATA → 硬盘

## 品牌推荐

- **一线**：海韵（Seasonic）、振华（Super Flower）、海盗船（Corsair）
- **性价比**：酷冷至尊（Cooler Master）、长城、鑫谷
- **避坑**：杂牌电源坚决不买，宁可其他配置降一档
