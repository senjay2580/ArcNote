# 考研英语二 · 阅读做题指南

> **核心方法论**：唐迟《考研英语阅读的逻辑》
> **补充来源**：研招网、考研帮、MBAChina 时间分配建议
>
> **我的个人痛点记录**（做题时反复犯的错 · 每次做题前先扫一眼）：
>
> 1. **语法 + 单词弱 → 阅读速度慢** — 根源问题
> 2. **看选项不认真** — 一定要**回文定位**，一个单词就可能挖坑
> 3. **不确定时马上拍板** — 文章一定要找到**对应的点**，不要不看其他选项
> 4. **没看完所有选项就选** — 除非**极度确定或其他选项极度确定是错**
> 5. **都不确定时凭感觉** — 此时**必须定位关键词回原文**
> 6. **题干没读清楚** — 有时审题本身就出错

## 一、考场时间安排（全题型 · 180 分钟）

### 1.1 推荐做题顺序 + 时间

> **原则**：作文先写（模板记忆最清晰）→ 阅读（最重分值）→ 新题型 → 翻译 → 完形（最后做性价比最低的）

| 顺序 | 题型 | 分值 | 用时 | 策略 |
|---|---|---|---|---|
| **① 先写** | 小作文 | 10 | **15 min** | 模板刚默完，趁热写 |
| **② 再写** | 大作文 | 15 | **25 min** | 图表描述 + 三段式 |
| **③ 主攻** | **阅读 Text 1–4** | **40** | **60–70 min** | **每篇 15–18 min**，核心战场 |
| ④ | 新题型 | 10 | **15 min** | 匹配题性价比高 |
| ⑤ | 翻译 | 15 | **20 min** | 拆分组合，逐句写 |
| ⑥ 最后 | 完形填空 | 10 | **15 min** | 不确定就猜，**不值得恋战** |
| | **机动时间** | | **10–20 min** | 检查涂卡 + 回头看标记题 |
| | **合计** | **100** | **180 min** | |

### 1.2 单篇阅读 15 分钟拆解

| 阶段 | 用时 | 动作 |
|---|---|---|
| **读题干** | 1.5 min | 读 5 个题干（**只读题干不看选项**），划关键词 |
| **读文章** | 6–7 min | 带着题干关键词读文章，**边读边标记定位** |
| **做题** | 5–6 min | 逐题回文定位 → 比对 4 选项 → 排除 → 选 |
| **检查** | 1 min | 有疑问的**标记**，最后统一回看 |

### 1.3 时间红线

- **单篇阅读超过 20 分钟** → **强制跳过**，猜一个先涂卡
- **完形超过 15 分钟** → 剩余的全选**同一个字母**（统计上 B/C 略高）
- **作文超过 40 分钟** → 后面题型全部会被压缩
- **一道题纠结超过 2 分钟** → 标记跳过，做完回来看

## 二、唐迟阅读方法论 · 三大核心原则

### 原则 1 · 先题干后文章

> 出题顺序 = 文章行文顺序

**操作**：
1. 先快速浏览 **5 个题干**（不看选项），每题划出 1–2 个**关键词**
2. 读文章时**带着关键词定位**
3. 读到哪个关键词就停下来做对应的题

**好处**：避免通读全文后忘了题目要什么，**精准打击**。

---

### 原则 2 · 细节服从主旨

> 每篇文章 = 主旨（论点）+ 细节（论据），**细节支持主旨**

当一个选项的表述和文章主旨**方向一致** → 优先选
当一个选项的表述和文章主旨**方向相反** → 大概率错

**实战用法**：不确定细节题答案时，**选和全文核心观点一致的那个**。

---

### 原则 3 · 反复强调的是重点，重点是考点

如果一个概念 / 词 / 态度在文中**出现 ≥ 2 次** → **极大概率是考点**
串联 5 个题干，**被反复提及的词** → **就是文章主旨词**

---

## 三、唐迟六大题型解法

### 题型 1 · 中心主旨题

**识别标志**：`mainly about` / `discuss` / `the best title` / `the purpose`

**解法**：

1. **串联各段首句** → 看反复出现的关键词
2. 找**"问题—回答"结构**的独句段
3. 首段末句 + 末段首句 = 主旨锁定区

**干扰项特征**：
- 只涉及部分段落（以偏概全）
- 过于宽泛，超出文章范围
- 文中从未提及的新概念

---

### 题型 2 · 词汇题

**识别标志**：`most probably means` + 行号

**解法**：

1. **有转折词**（but / however / yet）→ 找**反义词**
2. **无转折词** → 找**同义词**
3. **分号冒号** 后面的内容 = **对前面的解释**，即答案
4. **代词**优先指代主题词

**干扰项特征**：

- **字面意思**（考研从不考字面意思）
- 原文出现过的其他词（声东击西）

**铁律**：**词汇题的正确答案可以是原词的某个意思**，不一定非要改写。

---

### 题型 3 · 例证题

**识别标志**：`example` / `story` / `case` / `illustrate` / `demonstrate` / `mention … to`

**解法**：
> **例子本身不是答案，例子支持的观点才是答案**

1. 定位例子在原文的位置
2. **向上**找论点句（通常在例子前 1–2 句）
3. 如果上方没有 → **向下**找（在例子后 1 句）
4. 选项和**论点一致** = 正确

**干扰项特征**：
- **就事论事**：只描述例子内容本身，没有抽象到观点层面

---

### 题型 4 · 细节题（出现最多 · 约占 50%）

**识别标志**：`what` / `how` / `why` / `which` / `according to` + 具体信息

**解法**：
1. **题干关键词回文定位**（专有名词 / 时间 / 数字 / 大写字母）
2. **定位好的句子就是答案所在**
3. 注意**因果连词**（because / since / as / so that）周围常出答案
4. 注意**否定词和限定词**（not / only / never / all / some）

**干扰项特征**：
- **偷换概念**：把关键限定词悄悄改了（some → all）
- **无中生有**：原文根本没提
- **张冠李戴**：把 A 的特征安到 B 头上
- **强加因果**：两件事原文并列，选项硬说成因果

**⚠️ 我的个人提醒**：
- **一定要回文定位**，不要凭印象选
- **所有 4 个选项都要看完**，不要看到一个像就选
- **一个单词就可能是坑**：`some → all`、`may → must`、`before → after`

---

### 题型 5 · 作者态度题

**识别标志**：`attitude` / `deem` / `consider` / `tone` / `view`

**解法**：
1. **看形容词和副词** — 它们暴露态度
2. **看转折处** — 转折后才是作者真正态度
3. **看情态动词** — `should / must` = 强态度，`may / might` = 弱态度
4. 选项归类后排除：先分成**正面 / 负面 / 中立**三类

**永远不选的选项**：
- `indifferent`（漠不关心）— 作者写了一篇文章不可能漠不关心
- `confused`（困惑）— 作者不可能自己困惑
- `biased`（偏见）— 考研文章都来自正规期刊
- `prejudiced`（有偏见）— 同上
- `subjective`（主观）— 通常不选

---

### 题型 6 · 推理判断题

**识别标志**：`infer` / `imply` / `suggest` / `conclude` / `learn from`

**解法**：

1. **正确答案 = 原文的同义改写**（不是脑补推理）
2. **语气缓和的选项**更可能对：`may / probably / likely / tend to`
3. **绝对化选项**大概率错：`only / never / all / must / completely`

**干扰项特征**：
- **过度推理**：原文没说那么远
- **正反混淆**：作者说好，选项说坏

## 四、唐迟十大黄金法则

> **来源**：《考研英语阅读的逻辑》

| # | 法则 | 解释 |
|---|---|---|
| **1** | **时间相反则一切相反** | 过去 vs 现在 态度 / 做法常相反 |
| **2** | **少数派原则** | 文中提到少数人的观点，后文常被反驳 |
| **3** | **真题答案说一不二** | 正确答案一定能在原文找到**唯一依据** |
| **4** | **正确答案"路见不平"** | 正确答案紧扣原文，干扰项"拔刀相助"过度延伸 |
| **5** | **主旨题风格趋向文艺化** | 近年 best title 选项越来越隐喻 |
| **6** | **代词优先指代主题词** | it / they / this 优先回指文章核心概念 |
| **7** | **词汇题原词可为正确答案** | 不一定非要同义替换 |
| **8** | **内容关系只有两种：相同或相反** | 帮助判断逻辑走向 |
| **9** | **答案需比较得出** | 不是选"对的"，是选"最对的" |
| **10** | **感情色彩是阅读重点** | 积极 / 消极 / 中立决定全文方向 |

## 五、干扰项识别 · 五大特征（做题时逐个排查） 

| # | 干扰手段 | 如何识别 | 例子 |
|---|---|---|---|
| ① | **无中生有** | 原文**完全没提**这件事 | 原文讲教育，选项突然冒出"军事" |
| ② | **偷换概念** | 关键词**被悄悄替换** | `some students` → `all students` |
| ③ | **以偏概全** | 个别案例**被当成整体结论** | 一个公司的案例 → "所有企业都如此" |
| ④ | **强加因果** | 原文是并列/时间关系，选项**硬说成因果** | "A 发生了，B 也发生了" → "A 导致了 B" |
| ⑤ | **正反混淆** | 态度或结论方向**取反** | 作者支持 → 选项说反对 |

**做题流程**：看到一个选项，脑子里过一遍"这 5 条它犯了哪条"。

## 我的做题纪律清单（每次做阅读前默读一遍）

```
□ 1. 先读题干划关键词，再读文章
□ 2. 读文章时标记题干关键词出现的位置
□ 3. 做每道题时 → 必须回文定位 → 找到原文依据
□ 4. 看完所有 4 个选项再选，除非其他 3 个都 100% 能排除
□ 5. 不确定 → 定位题干关键词 → 回原文读上下 2 句
□ 6. 纠结 A 和 B → 看哪个更"贴原文"，而不是"更合理"
□ 7. 纠结超过 2 分钟 → 标记 → 跳过 → 做完回来
□ 8. 做完一篇 → 快速检查 5 题是否和文章主旨一致
□ 9. 审题！审题！审题！题干是问"正确的"还是"错误的"
□ 10. 单篇不超 20 分钟，超了就跳
```



###  根源诊断

速度慢 = **两大根因**：
1. **单词不认识** → 反复回看 → 单篇耗时 25+ 分钟
2. **长难句读不懂** → 脑子里反复翻译 → 卡住

### 对症方案

| 根因 | 对症训练 | 周期 |
|---|---|---|
| **单词弱** | 每天真题精读 1 篇，**不认识的词当场查 + 记到本子上**（不是背单词书） | 持续到考前 |
| **长难句慢** | 每天用唐静拆分法拆 3 个长难句（从翻译真题里选）| 4 周 |
| **回读多** | 训练**一遍读完不回头**：先读题干，带着问题读文章，强迫自己只前进 | 2 周 |
| **翻译腔** | 不要在脑子里把英文翻成中文再理解 → 训练**英文直接理解**：读简单外刊 | 8 周 |
