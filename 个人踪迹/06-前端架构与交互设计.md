# 前端架构与交互设计

## 一、技术栈选择

### 1.1 核心技术

| 技术 | 用途 | 特点 |
|------|------|------|
| Thymeleaf | 服务端模板 | 与 Spring 深度集成 |
| HTMX | 动态交互 | 无需编写 JavaScript |
| Leaflet.js | 地图可视化 | 轻量级、插件丰富 |
| 原生 JavaScript | 前端逻辑 | 无框架依赖 |

### 1.2 设计理念

```
传统 SPA:  服务端 API → JSON → 前端渲染
Reitti:    服务端渲染 → HTML 片段 → HTMX 替换
```

**优势**:
- 减少前端复杂度
- SEO 友好
- 首屏加载快
- 服务端控制渲染逻辑

## 二、HTMX 动态交互

### 2.1 时间线动态加载

```html
<div class="timeline-container"
     hx-get="/timeline/content/range"
     hx-trigger="dateChanged from:body"
     hx-vals="js:{startDate: getTimelineParams().startDate, 
                  endDate: getTimelineParams().endDate, 
                  timezone: getTimelineParams().timezone}"
     hx-indicator="#loading-indicator">
    <div id="loading-indicator" class="timeline-entry">Loading...</div>
</div>
```

**HTMX 属性解析**:
- `hx-get`: 发送 GET 请求到指定 URL
- `hx-trigger`: 触发条件 (自定义事件 `dateChanged`)
- `hx-vals`: 动态请求参数 (JavaScript 表达式)
- `hx-indicator`: 加载指示器元素

### 3.2 访问标记绘制

```javascript
function updateMapFromTimeline() {
    const bounds = L.latLngBounds();
    
    // 清除现有标记
    map.eachLayer(layer => {
        if (!layer._url) { // 保留瓦片图层
            map.removeLayer(layer);
        }
    });
    
    // 按坐标分组访问
    const placeGroups = new Map();
    
    timelineEntries.forEach(entry => {
        const lat = parseFloat(entry.dataset.lat);
        const lng = parseFloat(entry.dataset.lng);
        const coordKey = `${lat.toFixed(6)},${lng.toFixed(6)}`;
        
        if (!placeGroups.has(coordKey)) {
            placeGroups.set(coordKey, {
                lat, lng,
                totalDurationMs: 0,
                visits: [],
                place: { name: entry.querySelector('.place-name').textContent }
            });
        }
        
        // 累计停留时间
        const group = placeGroups.get(coordKey);
        group.totalDurationMs += parseDurationText(entry.querySelector('.entry-duration').textContent);
        group.visits.push({ id: entry.dataset.id });
    });
    
    // 绘制标记
    placeGroups.forEach((group) => {
        const { lat, lng, totalDurationMs, place } = group;
        
        // 根据停留时间计算圆圈半径 (对数缩放)
        const durationHours = totalDurationMs / (1000 * 60 * 60);
        const logScale = Math.log(1 + durationHours) / Math.log(1 + 24);
        const radius = Math.min(100, Math.max(15, 15 + logScale * 85));
        
        // 中心点标记
        L.circleMarker([lat, lng], {
            radius: 5,
            fillColor: lightenHexColor(color, 20),
            color: '#fff',
            weight: 1,
            opacity: 1,
            fillOpacity: 0.8
        }).addTo(map);
        
        // 范围圆圈
        const circle = L.circle([lat, lng], {
            color: color,
            fillColor: lightenHexColor(color, 20),
            fillOpacity: 0.1,
            radius: radius
        }).addTo(map);
        
        // 工具提示
        circle.bindTooltip(`
            <div class="visit-title">${place.name}</div>
            <div class="visit-description">
                ${group.visits.length} visits - Total: ${humanizeDuration(totalDurationMs)}
            </div>
        `);
        
        bounds.extend([lat, lng]);
    });
    
    return bounds;
}
```

### 3.3 原始位置数据加载器

```javascript
class RawLocationLoader {
    constructor(map, userSettings, fitToBoundsConfig) {
        this.map = map;
        this.userSettings = userSettings;
        this.fitToBoundsConfig = fitToBoundsConfig;
        this.userConfigs = [];
        this.polylines = new Map();
    }
    
    init(userConfigs) {
        this.userConfigs = userConfigs;
    }
    
    async loadForDateRange(autoUpdateMode, forceReload) {
        for (const config of this.userConfigs) {
            const response = await fetch(
                `${config.url}?startDate=${startDate}&endDate=${endDate}&timezone=${timezone}`
            );
            const points = await response.json();
            
            this.drawPolyline(config, points);
        }
    }
    
    drawPolyline(config, points) {
        // 移除旧的折线
        if (this.polylines.has(config.url)) {
            this.map.removeLayer(this.polylines.get(config.url));
        }
        
        // 创建新的折线
        const latLngs = points.map(p => [p.latitude, p.longitude]);
        const polyline = L.polyline(latLngs, {
            color: config.color,
            weight: 3,
            opacity: 0.7
        }).addTo(this.map);
        
        this.polylines.set(config.url, polyline);
    }
    
    setSelectedTimeRange(startTime, endTime) {
        // 高亮显示选中时间范围的路径
        return this.highlightTimeRange(startTime, endTime);
    }
    
    clearSelectedTimeRange() {
        // 清除高亮
        this.clearHighlight();
    }
}
```

class DatePicker {
    constructor(containerId, options) {
        this.container = document.getElementById(containerId);
        this.options = {
            daysToShow: 14,
            prefetchDays: 25,
            allowRangeSelection: true,
            singleDateMode: true,
            dateFormat: 'YYYY-MM-DD',
            ...options
        };
        this.selectedRange = { startDate: null, endDate: null };
        this.listeners = {};
    }
    
    on(event, callback) {
        if (!this.listeners[event]) {
            this.listeners[event] = [];
        }
        this.listeners[event].push(callback);
    }
    
    emit(event, data) {
        if (this.listeners[event]) {
            this.listeners[event].forEach(cb => cb(data));
        }
    }
    
    setSelectedRange(startDate, endDate) {
        this.selectedRange = { startDate, endDate };
        this.render();
        this.emit('selectionChange', this.selectedRange);
    }
    
    render() {
        // 渲染日期选择器 UI
        // 支持日/月/年三种时间粒度
        // 支持滚轮切换粒度
    }
}

## 五、实时更新 (SSE)

### 5.1 Server-Sent Events 连接

```javascript
let eventSource = null;

function connectSSE() {
    eventSource = new EventSource('/api/v1/sse/subscribe');
    
    eventSource.onmessage = function(event) {
        const data = JSON.parse(event.data);
        handleSSEMessage(data);
    };
    
    eventSource.onerror = function() {
        showMessage(locale.sse.error);
        // 自动重连
        setTimeout(connectSSE, 5000);
    };
}

function handleSSEMessage(data) {
    switch (data.type) {
        case 'NEW_VISIT':
            // 刷新时间线
            document.body.dispatchEvent(new CustomEvent('dateChanged'));
            break;
        case 'NEW_TRIP':
            // 更新地图
            rawLocationLoader.loadForDateRange(true, true);
            break;
        case 'PROCESSING_COMPLETE':
            showNotification('Processing complete');
            break;
    }
}
```

### 5.2 自动更新模式

```javascript
let autoUpdateMode = false;

function enableAutoUpdate() {
    autoUpdateMode = true;
    document.body.classList.add('auto-update-mode');
    
    // 显示实时时间
    autoUpdateTimeInterval = setInterval(() => {
        document.getElementById('auto-update-time').textContent = 
            new Date().toLocaleTimeString();
    }, 1000);
    
    // 自动刷新数据
    autoUpdateTimer = setInterval(() => {
        rawLocationLoader.loadForDateRange(true, true);
    }, 30000);
}

function disableAutoUpdate() {
    autoUpdateMode = false;
    document.body.classList.remove('auto-update-mode');
    clearInterval(autoUpdateTimeInterval);
    clearInterval(autoUpdateTimer);
}
```

## 六、照片集成

### 6.1 照片客户端

```javascript
class PhotoClient {
    constructor(map, enabled) {
        this.map = map;
        this.enabled = enabled;
        this.photoMarkers = [];
    }
    
    async updatePhotosForRange(startDate, endDate, timezone) {
        if (!this.enabled) return;
        
        // 清除现有照片标记
        this.clearPhotoMarkers();
        
        // 获取照片数据
        const response = await fetch(
            `/api/v1/photos?startDate=${startDate}&endDate=${endDate}&timezone=${timezone}`
        );
        const photos = await response.json();
        
        // 添加照片标记
        photos.forEach(photo => {
            const marker = L.marker([photo.latitude, photo.longitude], {
                icon: this.createPhotoIcon(photo.thumbnailUrl)
            }).addTo(this.map);
            
            marker.on('click', () => this.showPhotoModal(photo));
            this.photoMarkers.push(marker);
        });
    }
    
    showPhotoModal(photo) {
        // 显示全屏照片查看器
        const modal = document.createElement('div');
        modal.className = 'photo-modal';
        modal.innerHTML = `
            <img src="${photo.fullUrl}" alt="${photo.filename}">
            <button class="close-btn">&times;</button>
        `;
        document.body.appendChild(modal);
        
        // 键盘导航
        document.addEventListener('keydown', this.handleKeyNavigation);
    }
}
```

function restoreTimelineState() {
    const timelineHidden = localStorage.getItem('timelineHidden');

    }
}

## 九、学习要点

### 9.1 HTMX 优势

1. **简化前端**: 无需复杂的前端框架
2. **渐进增强**: 基础功能不依赖 JavaScript
3. **服务端控制**: 渲染逻辑在服务端
4. **减少代码**: 用 HTML 属性替代 JavaScript

### 9.2 地图可视化技巧

1. **对数缩放**: 停留时间用对数缩放显示
2. **颜色区分**: 不同用户使用不同颜色
3. **聚合显示**: 相同位置的访问聚合显示
4. **动态加载**: 按需加载原始位置数据

### 9.3 用户体验优化

1. **状态持久化**: 使用 localStorage 保存用户偏好
2. **实时反馈**: SSE 推送实时更新
3. **键盘导航**: 照片查看器支持键盘操作
4. **加载指示**: HTMX 自动显示加载状态
