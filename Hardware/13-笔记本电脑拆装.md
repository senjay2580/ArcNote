# 13 - 笔记本电脑拆装

## 笔记本 vs 台式机的区别

| 对比 | 台式机 | 笔记本 |
|------|--------|--------|
| 可换部件 | 几乎所有 | 通常只有内存、SSD、WiFi卡、电池 |
| CPU/GPU | 独立可拆 | 焊在主板上（BGA） |
| 散热结构 | 塔式/水冷 | 铜管+离心风扇 |
| 拆机难度 | 简单 | 需要耐心和正确工具 |
| 螺丝 | 大螺丝，几种规格 | 小螺丝，型号多 |

## 拆机前准备

### 必备工具
- **十字螺丝刀**：PH0 或 PH1（小号十字）
- **翘片/撬棒**：塑料材质，撬开卡扣用（吉他拨片也行）
- **防静电手环**（可选但推荐）
- **螺丝收纳盒**：不同位置的螺丝长度不同，要分类放！
- **手机/相机**：每一步拍照记录！

### 拆机前必做
1. **关机并拔掉电源线**
2. **长按电源键 15 秒**（释放残余电力）
3. **准备好工具和收纳盒**
4. **在干燥、光线充足的桌面上操作**
5. **拍一张完整的底部照片**（记录螺丝位置）

## 通用拆机步骤（以联想 Y7000P 为例）

### 第一步：拆后盖
1. 翻转笔记本，底面朝上
2. 拧下所有底部螺丝（Y7000P 约 10-12 颗）
   - **注意**：有些螺丝不同长度！按位置分类放好
   - 有些螺丝可能隐藏在脚垫或贴纸下面
3. 用翘片从缝隙处插入，沿边缘划开卡扣
   - **不要用金属工具！** 会划伤
   - 听到"咔咔"声是卡扣打开的声音，正常
4. 轻轻掀开后盖

### 第二步：断开电池
**最重要的一步！拆任何东西之前先断电池！**
1. 找到电池排线连接器（通常在电池旁边）
2. 轻轻拔起排线接头（向上抬起，不是向外拉）

> 不断电池就操作 = 短路风险 = 可能烧主板！

## 常见可操作项目

### 更换/加装内存
1. 断电池
2. 扳开内存两侧的金属卡扣 → 内存弹起 45°
3. 斜着抽出旧内存
4. 新内存 45° 斜插入 → 按下去卡住

### 更换/加装 M.2 SSD
1. 断电池
2. 拧下 SSD 固定螺丝
3. SSD 弹起，斜着抽出
4. 新 SSD 斜插入 → 按下 → 拧螺丝

### 更换 WiFi 网卡
1. 断电池
2. **拔下两根天线线（黑色和白色）**：用指甲或小镊子轻轻竖直拔起
   - 这两根就是 u.FL 天线接头
   - 你的 Y7000P WiFi 问题很可能就是这里松了
3. 拧下固定螺丝
4. 抽出旧网卡，插入新的
5. 接回天线线（黑接黑、白接白，按下去"咔"一声）
6. 拧螺丝

### 清灰 + 换硅脂
1. 断电池
2. 拆掉散热模组（拧下铜管上所有螺丝，注意对角线顺序拆）
3. 用软毛刷清理风扇叶片上的灰
4. 用压缩空气吹散热鳍片
5. 用酒精棉/无尘布擦掉 CPU 和 GPU 上的旧硅脂
6. 涂新硅脂（米粒大小）
7. 按对角线顺序装回散热模组（先轻拧，再逐步拧紧）

> **散热螺丝有编号！** 按 1-2-3-4 的顺序拧，确保压力均匀

### 更换电池
1. 断开电池排线
2. 拧下电池固定螺丝（通常 4-6 颗）
3. 取出旧电池（如果有胶粘着，慢慢撬）
4. 放入新电池，拧螺丝，接排线

## 装回注意事项

1. **先接电池排线**
2. 检查所有排线有没有被压住
3. 对齐后盖，从一个角开始扣上，沿边缘按压
4. 拧回所有螺丝（对照你拍的照片！）
5. 开机测试

## 拆机禁忌

- **不要** 在地毯/床上拆机（静电）
- **不要** 用金属工具撬卡扣
- **不要** 大力拉排线
- **不要** 忘记拔电池就开始操作
- **不要** 把螺丝混在一起（不同位置长度不同）
- **不要** 硬掰掰不动的东西（可能还有隐藏螺丝）

## 你现在可以做的事

根据你的 Y7000P WiFi 问题，建议：
1. 按上面步骤拆开后盖
2. 检查 WiFi 网卡（AX211）的 M.2 螺丝是否松动
3. 检查两根天线接头（u.FL）是否牢固
4. 顺便清灰 + 换硅脂（已经 2-3 年了）
