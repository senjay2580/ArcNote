# 高效刷题方法

## 核心流程：每天晚自习分三块

### 一、回忆当天所学（15-20分钟）

- **不是**简单看笔记/教材
- 必须**脱离所有材料**，在脑海中按老师讲课顺序完整过一遍
- 如果实在回忆不起来：先快速看一遍笔记，然后**合上笔记再回想**
- 理科（数学/物理/化学）：拿一张空白草稿纸，想到什么写什么（公式、图形、推导）

### 二、限时刷题

- 用**秒表计时**，模拟考试节奏
- 初期可以放松要求，逐步提高到考试速度甚至更快
- 做题过程中**不看材料、不问人、不被打断**

### 三、立即纠错（最关键）

- 做完题**马上**纠错，不要隔天（否则忘了自己的思路）
- 纠错的核心：**精确定位错因**——具体到哪一步写错的，不能笼统说"不会做"
- 把错误原因写在卷子旁边
- 搞懂正确解法后，**不看答案独立再做一遍**，并能把思路讲出来，才算真正搞懂

### 四、间隔复习错题

| 时间节点 | 说明 |
|---------|------|
| 第二天 | 最近的复习，要频繁 |
| 周末 | 巩固一周错题 |
| 半个月后 | 中期回顾 |
| 一个月后 | 长期巩固 |

- 每复习一遍在题目旁边**加一个星号**，追踪复习次数
- 有时间 → 完整独立重做
- 没时间 → 不看答案回忆思路，能讲明白即可

---

## 预期效果

随着习惯养成，做题速度越来越快，写作业时间越来越少，复习时间越来越多。最终从每天两三节课写作业，变成一节课甚至提前做完，剩余时间全部用于复习。
