# 03 - 主板详解

## 主板是什么？

主板（Motherboard）是电脑的"骨架"和"神经系统"，所有硬件都插在主板上，通过主板上的电路互相通信。

## 主板上的关键部位（从上到下）

```
┌─────────────────────────────────────────────┐
│  [后置I/O接口面板]                            │
│  USB / HDMI / 网口 / 音频 / WiFi天线          │
├─────────────────────────────────────────────┤
│                                              │
│  [CPU插槽]          [供电模块VRM]             │
│   Socket区域         (CPU旁边一排)            │
│                                              │
│  [内存插槽] x2/x4   [M.2 SSD插槽]           │
│   DDR4/DDR5          NVMe固态硬盘             │
│                                              │
│  [PCIe x16插槽]     ← 显卡插这里             │
│  [PCIe x1 插槽]     ← 网卡/声卡等            │
│                                              │
│  [芯片组]            [SATA接口] x4/x6        │
│   南桥散热片下面      传统硬盘/SATA SSD        │
│                                              │
│  [前面板接针]        [风扇接口]               │
│   开机键/USB等        CPU_FAN / SYS_FAN       │
│                                              │
│  [24pin主板供电]     [8pin CPU供电]           │
│                                              │
└─────────────────────────────────────────────┘
```

## 核心部件详解

### 1. CPU 插槽（Socket）
- **Intel LGA 1700**：12/13/14代酷睿（触点式）
- **Intel LGA 1851**：Core Ultra 200系列
- **AMD AM5**：锐龙7000/9000系列（触点式）
- **AMD AM4**：锐龙1000~5000系列（针脚式）

> CPU 和主板的 Socket 必须一致，否则物理上插不进去

### 2. 芯片组（Chipset）
芯片组决定主板的功能和扩展能力：

| Intel 芯片组 | 定位 | 特点 |
|-------------|------|------|
| H610 | 入门 | 不能超频，接口少 |
| B660/B760 | 主流 | 性价比高，不能超频 |
| Z690/Z790 | 高端 | 可超频，接口最多 |

| AMD 芯片组 | 定位 | 特点 |
|-----------|------|------|
| A620 | 入门 | 不能超频 |
| B650 | 主流 | 性价比高，可超频 |
| X670/X870 | 高端 | PCIe 5.0，接口最多 |

### 3. 内存插槽（DIMM Slots）
- 通常 2 或 4 个插槽
- DDR4 和 DDR5 插槽有防呆缺口，互不兼容
- 双通道：插 2 条内存时，优先插 **第2和第4 槽**（隔一个插）

### 4. PCIe 插槽
- **PCIe x16**：给显卡用的，最宽最长
- **PCIe x4/x1**：给网卡、声卡、采集卡等
- PCIe 版本：3.0 → 4.0 → 5.0（每代带宽翻倍）

### 5. M.2 插槽
- 安装 NVMe SSD 的小插槽
- 常见规格：M.2 2280（22mm宽，80mm长）
- 有的主板带散热片覆盖

### 6. SATA 接口
- 连接传统机械硬盘（HDD）和 SATA SSD
- 速度上限 6Gbps（约 550MB/s）
- 用 SATA 数据线连接

### 7. 供电模块（VRM）
- CPU 插槽周围的一排小方块组件
- 负责给 CPU 提供稳定的电压
- 相数越多（如 12+2 相）供电越稳，越能带高端 CPU

### 8. 前面板接针（Front Panel Header）
- 连接机箱的开机按钮、重启按钮、电源灯、硬盘灯
- 这是装机最"烦人"的一步，线很细要对照说明书插

### 9. 后置 I/O 面板
主板背面能看到的接口：
- USB 2.0 / 3.0 / 3.2 / Type-C
- HDMI / DisplayPort（核显输出）
- RJ45 网口（2.5G 越来越普及）
- 3.5mm 音频接口
- WiFi 天线接口（带无线网卡的主板）

## 主板尺寸规格

| 规格 | 尺寸 | 特点 |
|------|------|------|
| **ATX** | 305×244mm | 标准大板，扩展性最好 |
| **Micro-ATX (mATX)** | 244×244mm | 主流小板，性价比高 |
| **Mini-ITX** | 170×170mm | 迷你机箱用，扩展少 |
| **E-ATX** | 305×330mm | 服务器/发烧级 |

> 机箱要兼容主板尺寸，ATX 机箱能装 ATX 和更小的板子


## 你的电脑的主板

联想 Y7000P IRH8 的主板：
- 笔记本专用主板，非标准尺寸
- CPU（i7-13620H）焊死在主板上
- 2 个 SO-DIMM 内存插槽（可换）
- 1-2 个 M.2 SSD 插槽（可换）
- 1 个 M.2 WiFi 卡插槽（你的 AX211 就插在这里）
