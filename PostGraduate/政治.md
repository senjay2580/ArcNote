#### （一）单选题（1-16 题，16 分）：套路 + 猜题 + 陷阱

##### 1. 出题核心套路（固定模块 + 固定考法）

- 模块固定：**1-4 马原、5-8 毛中特、9-12 史纲、13-14 思修、15-16 时政**

- 考法固定：

  1. 概念定义 / 本质 / 核心 / 根本 / 标志 / 首次 / 最（如 “根本任务”“核心要义”“首次提出”）
  2. 原理对应（马原）、会议 / 著作 / 时间 / 人物（史纲）、政策依据（毛中特）、道德 / 法治原则（思修）
  3. 领导人原话、中央文件原文、时政热点原话（直接挖空）

  

- 选项设计：**3 错 1 对**，错误选项多为：概念偷换、范围扩大 / 缩小、绝对化、张冠李戴、因果倒置

##### 2. 单选题猜题 / 蒙题黄金法则（正确率≥70%）

1. **绝对化表述必错**：出现 “唯一、完全、全部、根本、只有、仅仅、必然、永远、消除、杜绝” 等，90% 排除
2. **最核心 / 根本 / 本质 / 标志**：优先选**最根本、最核心、总依据、总遵循、灵魂、主线**类表述（如 “总依据是初级阶段”）
3. **马原单选**：优先选**唯物论 / 辩证法 / 认识论**的正确原理，唯心 / 形而上学直接排除
4. **史纲单选**：**时间、会议、著作、人物**一一对应，错配直接排除
5. **毛中特 / 思修**：**领导人原话、中央文件原文**必对，偏离原文的修饰 / 改写多错
6. **时政单选**：**中国立场、正面表述、积极意义、合作共赢**必对，负面 / 对立 / 霸权表述必错
7. **二选一纠结**：选**更全面、更根本、更符合主流价值观、更贴近教材原文**的选项

##### 3. 常见陷阱（避坑）

- 偷换概念：如 “根本动力”≠“直接动力”，“基本矛盾”≠“主要矛盾”
- 范围错配：如 “社会主义初级阶段”≠“整个社会主义”，“全体人民”≠“部分人民”
- 绝对化：如 “完全解决”“彻底消除”“唯一途径”
- 张冠李戴：如把毛泽东的话安给邓小平，把十九大的内容安给十八大

#### （二）多选题（17-33 题，34 分）：套路 + 猜题 + 陷阱（最难、拉分核心）

##### 1. 出题核心套路（固定模块 + 固定考法）

- 模块固定：**17-21 马原、22-26 毛中特、27-29 史纲、30-31 思修、32-33 时政**

- 考法固定：

  1. **原理多维度、原因多方面、意义多层面、条件多要素**（多选核心：多角度、多方面）
  2. 正确选项多为**2-4 选**，极少 1 选（1 选基本是陷阱）
  3. 题干多为 “**体现了 / 说明了 / 原因是 / 意义在于 / 正确的有**”

  

- 选项设计：**正确选项≥2，错误选项为：片面、绝对、无关、偷换、反向**

##### 2. 多选题猜题 / 蒙题黄金法则（正确率≥60%）

1. **绝对化表述必排**：同单选，“唯一、完全、全部、根本、只有、仅仅、必然、永远、消除、杜绝” 直接排除
2. **正面 / 积极 / 符合主流价值观必选**：如 “有利于、促进、保障、推动、完善、增强、提升”
3. **中国立场、人民立场、党的领导、社会主义方向必选**（毛中特 / 史纲 / 思修 / 时政）
4. **马原多选**：**辩证法（联系 / 发展 / 矛盾）、认识论（实践 / 认识 / 真理）、唯物史观（人民群众）** 全选概率高
5. **史纲多选**：**党的正确决策、统一战线、人民群众、民族独立、人民解放**必选
6. **时政多选（32-33）**：**中国倡议、国际合作、共赢、开放、包容、公平正义**全选概率极高（4 选常见）
7. **排除法优先**：先排绝对化、无关、错误，剩下全选；纠结时**宁多勿少**（多选不倒扣分）
8. **肖四肖八规律**：**马原 2-3 选、毛中特 3-4 选、史纲 2-3 选、思修 2-3 选、时政 3-4 选**

##### 3. 多选题高频陷阱（避坑）

- 无关选项：选项本身正确，但与题干无关（最易错）
- 片面选项：只讲一方面，忽略另一方面（如只讲个人不讲集体）
- 偷换概念：如 “主要矛盾”≠“基本矛盾”，“共同富裕”≠“同步富裕”
- 绝对化：如 “完全实现”“彻底解决”“唯一标准”
- 反向选项：与题干要求相反（如题干问 “原因”，选项是 “结果”）

### 三、日常备考 + 应试训练：快速熟练模式，拿高分（选择题 + 大题）

#### （一）日常备考：基础 + 技巧 + 刷题（3 阶段）

##### 阶段 1：基础夯实（3-6 月）

- 教材：**5 本官方马工程教材 + 肖秀荣精讲精练**，重点理解马原原理、史纲时间线、毛中特核心概念
- 视频：徐涛强化班，搭建知识框架，**不背，只理解**
- 刷题：肖秀荣 1000 题，**刷 1 遍，错题标注，不纠结正确率**

##### 阶段 2：强化提升（7-10 月）

- 技巧：腿姐技巧班，**掌握选择题绝对化排除、关键词法、多选题排除法**
- 刷题：1000 题**刷 2-3 遍，错题复盘，整理错题本**，重点攻克马原、毛中特
- 框架：梳理**马原原理表、史纲时间线、毛中特会议 / 政策表、思修核心考点表**

##### 阶段 3：冲刺押题（11-12 月）

- 肖八：**选择题刷 3 遍，全对，错题背下来；大题看思路，不背**
- 肖四：**选择题全对，大题全背（4 套卷，每道题背熟）**
- 时政：看肖秀荣时政小册子，**重点记中国立场、国际合作、领导人讲话**
- 模拟：用真题 + 肖四肖八模拟，**控制时间，选择题 30 分钟，大题 100 分钟**

#### （二）应试训练：选择题 + 大题，快速熟练模式

##### 1. 选择题应试模式（30 分钟搞定，正确率≥85%）

- 步骤 1：**读题干，抓关键词**（根本、核心、本质、首次、最、原因、意义）
- 步骤 2：**排绝对化、无关、错误选项**（先排错，再选对）
- 步骤 3：**单选题：1 选 1 对，纠结选最根本；多选题：宁多勿少，正面全选**
- 步骤 4：**涂卡，检查，不纠结**（单选 1 分钟 / 题，多选 2 分钟 / 题）

##### 2. 大题应试模式（100 分钟搞定，得分率≥80%）

- 模板（通用）：

  总 - 分 - 总

  1. 总：**点题，写出核心原理 / 观点**（1-2 句）
  2. 分：**结合材料，分点论述，原理 + 材料分析**（3-4 点，每点原理 + 材料）
  3. 总：**总结，升华，方法论 / 意义 / 青年怎么做**（1-2 句）

  

- 分题型模板：

  - 马原（34）：**原理句 + 材料分析 + 方法论**（如 “认识具有相对性，材料中……，因此要……”）
  - 毛中特（35）：**地位 + 意义 + 措施 + 材料**（如 “全面深化改革是……，材料中……，因此要……”）
  - 史纲（36）：**背景 + 过程 + 意义 + 经验**（如 “抗战胜利是……，材料中……，启示我们……”）
  - 思修（37）：**精神内涵 + 青年怎么做 + 材料**（如 “治沙精神是……，青年要……”）
  - 当代（38）：**全球问题 + 中国方案 + 意义 + 合作**（如 “数字鸿沟是……，中国方案是……，有利于……”）

  

- 技巧：**肖四大题答案改写，结合材料，分点清晰，字迹工整**
