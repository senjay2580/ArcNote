# 14 - 硬件选购指南

## 选购原则

1. **明确需求**：先确定用途，再定预算
2. **均衡搭配**：不要一个部件特别好，其他拉胯（木桶效应）
3. **兼容性第一**：CPU 和主板、内存代数、电源功率都要匹配
4. **不追最新**：上一代产品性价比通常最高

## 按用途推荐配置方向

### 办公/上网/影音
```
CPU:    i3 / Ryzen 5（带核显）
内存:   16GB DDR4/DDR5
SSD:    512GB NVMe
显卡:   不需要独显（用核显）
电源:   350-450W
预算:   2000-3000 元
```

### 游戏主机（1080P 高画质）
```
CPU:    i5-13400/14400 或 Ryzen 5 7600
内存:   16-32GB DDR5
SSD:    1TB NVMe
显卡:   RTX 4060 / RX 7600
电源:   550-650W 金牌
预算:   5000-7000 元
```

### 游戏主机（2K 高画质）
```
CPU:    i5-14600K 或 Ryzen 7 7800X3D
内存:   32GB DDR5
SSD:    1-2TB NVMe
显卡:   RTX 4070 Ti SUPER / RX 7900 GRE
电源:   750-850W 金牌
预算:   8000-12000 元
```

### 生产力（视频剪辑/3D建模/AI）
```
CPU:    i7/i9 或 Ryzen 9
内存:   32-64GB DDR5
SSD:    2TB+ NVMe
显卡:   RTX 4070 Ti+ (CUDA加速)
电源:   850W+ 金牌
预算:   12000-20000+ 元
```

## 各部件选购要点

### CPU
- **游戏**：单核性能重要，Ryzen 7 7800X3D 游戏最强
- **生产力**：多核性能重要，核心数越多越好
- **不要买最新一代首发**：等几个月降价 + 修 bug

### 主板
- 不超频：B 系列（B760/B650）够用
- 要超频：Z/X 系列
- 看 VRM 供电相数（带高端 CPU 至少 8 相以上）
- 看接口数量：USB、M.2 插槽够不够用

### 内存
- 2025 年新装机建议 DDR5
- 买两条组双通道（2×16GB = 32GB）
- 关注频率和时序（DDR5 5600 CL36 是甜点）
- 同品牌同型号同批次最稳

### SSD
- 系统盘选有 DRAM 缓存的（性能稳定）
- PCIe 4.0 性价比最高（对日常使用和 3.0 差距不大）
- 品牌推荐：三星 990 EVO、致态 TiPlus7100、西数 SN770
- 看 TBW（总写入量）和质保年限

### 显卡
- **先确定显示器分辨率和目标帧率**，再选卡
- 不要买二手矿卡
- A卡（AMD）相同性能通常便宜些，但光追弱于 N卡
- 关注显存大小：2K 游戏建议 10GB+

### 电源
- **绝对不能省！** 劣质电源可能烧全套硬件
- 额定功率留 100-150W 余量
- 认准 80Plus 金牌以上
- 推荐品牌：海韵、振华、海盗船、酷冷至尊

### 机箱
- 确认支持主板尺寸（ATX/mATX/ITX）
- 确认显卡限长（高端卡 >30cm）
- 散热位：支持多少风扇、能装多大冷排
- 走线空间：背板深度 >2cm
- 前置 USB-C 是加分项

### 散热器
- 65W CPU：原装散热器勉强够用
- 高端 CPU（>100W）：塔式风冷或 240/360 水冷
- 推荐：利民 PA120、猫头鹰 NH-D15、Arctic Liquid Freezer

## 在哪里买？

| 渠道 | 优点 | 缺点 |
|------|------|------|
| **京东自营** | 正品保障、退换方便 | 价格略高 |
| **天猫官方旗舰店** | 正品、活动价格好 | 退换稍慢 |
| **拼多多百亿补贴** | 价格最低 | 部分店铺需甄别 |
| **电脑城实体店** | 能看实物 | 水深，容易被坑 |
| **闲鱼/二手** | 便宜 | 高风险，需要经验 |

> **建议新手**：京东自营 > 天猫旗舰店 > 拼多多百亿补贴

## 装机配置单检查清单

- [ ] CPU 和主板 Socket 匹配？
- [ ] 内存代数和主板匹配？（DDR4/DDR5）
- [ ] 电源功率够用？（CPU TDP + 显卡 TGP + 150W）
- [ ] 机箱装得下主板和显卡？
- [ ] CPU 没核显的话有独显吗？
- [ ] 散热器压得住 CPU 的功耗？
- [ ] M.2 插槽数量够用？
- [ ] 有没有遗漏的配件？（键鼠、显示器、散热硅脂）

## 实用网站/工具

- **装机模拟器**：PC Building Simulator（Steam 游戏，练手用）
- **兼容性检查**：PCPartPicker（英文）
- **性能天梯图**：搜索"CPU天梯图""显卡天梯图"
- **价格监控**：什么值得买、慢慢买（历史价格）
- **评测参考**：B站硬件评测频道（硬件茶谈、极客湾等）
