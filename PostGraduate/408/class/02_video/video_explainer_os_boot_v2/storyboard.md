# OS Boot Explainer Storyboard v2.2

| Time | Beat | Layout Mechanism |
|---|---|---|
| 0-8s | 启动是三棒接力 | Dark relay arena + moving control baton + handoff flash |

v2.2 的目标是继续减少“固定 PPT 母版感”：保留 408 教学清晰度，但让每段有自己的视觉动作、镜头重心和动画节奏。这个版本也会沉淀为 `video-generate` skill 的 HyperFrames 主模板。
