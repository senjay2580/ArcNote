# 小程序申请・认证 + 备案 流程与材料清单（企业版）

您好，为顺利完成您的小程序上线，需要您配合提供以下材料，并按流程完成认证与备案，我这边全程协助操作。

---

## 一、需要提供的材料

### 1. 企业基础信息
- 常用邮箱（用于接收官方通知）
- 联系电话、应急联系电话

### 2. 证件资料（彩色清晰照片）
- 营业执照照片
- 法人身份证正反面

### 3. 小程序基础信息
- 小程序名称
- 小程序头像
- 小程序简介（用途介绍）

---

## 二、办理流程（我来操作，您配合）

### 1. 提交认证
- 用您提供的资料提交微信认证，支付 **300元/年** 官方审核费用
- 审核期间请保持法人电话畅通，留意接听认证回访电话

### 2. 完善小程序信息
- 认证通过后，在后台填写小程序类目、功能、介绍等信息

### 3. 提交备案（国家要求必做）
- 录入营业执照地址、法人信息、联系方式、邮箱
- 选择服务类型，普通行业一般选「以上都不涉及」
- 负责人完成人脸核验

### 4. 审核与短信核验
- 提交后 **1–2个工作日** 初审
- 初审通过后，法人会收到工信部短信，请在 **24小时内** 点开链接完成核验
  - 操作步骤：勾选小程序 → 填手机号 → 验证码 → 身份证后六位

### 5. 审核通过，小程序正式上线

---

## 三、您需要配合的关键点

- 资料一次性提供齐全，避免反复修改耽误进度
- 认证与备案期间，法人电话保持畅通
- 收到工信部核验短信，务必 **24小时内** 完成

> 我这边全程协助提交与跟进，您只需配合提供材料和关键核验即可。

## 四、微信相关配置教程

### 微信支付

> 后台：https://pay.wechatpay.cn/

| 参数 | 说明 | 从哪里获取 |
|------|------|-----------|
| `appid` | 小程序 AppID | 微信公众平台 |
| `mchid` | 商户号 | 微信商户平台 |
| `apiv3_key` | APIv3 密钥（32位） | 商户平台 → 账户中心 → API安全 |
| `serial_no` | 商户证书序列号 | 申请证书时生成 |
| `private_key` | 商户私钥（.pem文件） | 申请证书时下载 |

> 待填写：mchid、apiv3_key、serial_no、private_key（申请商户号后补充）

---

### 微信客服

> 后台：https://work.weixin.qq.com/wework_admin/loginpage_wx

| 入口 | 产品名 | 本质 |
|------|--------|------|
| 企业微信后台 | 微信客服（独立产品） | 专门的客服工单系统，支持多坐席、排队、转接、工单记录 |
| 微信商户后台 | 公众号客服 / 微信客服 | 绑定在公众号/小程序下的客服消息，功能更基础 |

**小程序调用参数：**

| 参数 | 说明 | 从哪里获取 |
|------|------|-----------|
| `corpId` | 企业ID | 企业微信后台 → 我的企业 → 企业信息 |
| `clientId` | 客服账号ID | 企业微信后台 → 应用管理 → 微信客服 → 账号管理 → 接入渠道 |

> 待填写：corpId、clientId（向企业微信管理员获取）
